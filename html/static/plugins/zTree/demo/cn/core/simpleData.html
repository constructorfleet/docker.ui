    • 1、setting Configure信息说明

      • 必须Settings setting.data.simple 内的Properties，For more details, please see API in document的相关Contents

    • 2、treeNode NodesData说明

      • Simple模式的 JSON Data需要Use id / pId 表示Nodes的父子包含关系，如UseOtherPropertiesSettings父子关联关系请参考 setting.data.simple Notes 
        For example:：
var nodes = [
	{id:1, pId:0, name: "父Nodes1"},
	{id:11, pId:1, name: "子Nodes1"},
	{id:12, pId:1, name: "子Nodes2"}
];
      • 默认Expand的Nodes，请Settings treeNode.open Properties
      • 无子Nodes的父Nodes，请Settings treeNode.isParent Properties
      • OtherProperties说明请参考 API in document "treeNode NodesData详解"
