[ DocumentationPath: core/standardData.html ]

    • 1、setting Configure信息说明

      • 普通Use，无必须Settings的Parameters
      • 与显示相关的Contents请参考 API in document setting.view 内的Configure信息
      • name、children、title Equal Properties定义更改请参考 API in document setting.data.key 内的Configure信息

    • 2、treeNode NodesData说明

      • Standard的 JSON Data需要嵌套表示Nodes的父子包含关系 
        For example:：
var nodes = [
	{name: "父Nodes1", children: [
		{name: "子Nodes1"},
		{name: "子Nodes2"}
	]}
];
      • 默认Expand的Nodes，请Settings treeNode.open Properties
      • 无子Nodes的父Nodes，请Settings treeNode.isParent Properties
      • OtherProperties说明请参考 API in document "treeNode NodesData详解"
