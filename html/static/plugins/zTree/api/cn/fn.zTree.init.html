Function(obj, zSetting, zNodes)$.fn.zTree.init

Overview[ Dependency jquery.ztree.core Core js ]

zTree InitializeMethodology，Create zTree 必须Use此Methodology

1、Page需要进行 W3C Affirming，For example:：<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN"    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">。

2、Need to load first jquery-1.4.2.js 或Other更高Version的 jQuery 。

3、Needs Loading jquery-ztree.core-3.0.js，If需要用到 编辑Functions 或 checkbox / radio It needs to be loaded separately jquery-ztree.exedit-3.0.js 和 jquery-ztree.excheck-3.0.js 。

4、Needs Loading zTreeStyle.css and zTreeStyle Under Directory img Documentation。

5、If需要Use自定义IconPlease相应的Demo。

6、请AttentionSettings zTree 的ContainersStyles class="ztree"，of which "ztree" Here className，可以Based on需要随意Modify，别忘了Modify css 中CorrespondName就是了，YeahContainersIf需要IncreaseOther特殊Styles，可Based on自己的需要进行Modify。

Function Parameters说明

objjQuery Object

For presentation zTree 的 DOM Containers

zSettingJSON

zTree 的ConfigureData，具体Please “setting Configure详解”Medium各个Properties详细说明

zNodesArray(JSON) / JSON

zTree 的NodesData，具体Please “treeNode NodesData详解”Medium各个Properties详细说明

1、v3.x Support单独Add一个Nodes，即If只新增一个Nodes，You don\'t have to wrap it in an array

2、If需要Step Load根Nodes，可以Settings为 null 或 [ ]

3、UseSimpleData模式，Please setting.data.simpleData 内的Properties说明

Back值JSON

zTree Object，ProvisionOperation zTree 的各种Methodology，YeahPass js Operation zTree 来说必须Pass此Object

IfNo need自行设定全局变量Save，可以Utilization $.fn.zTree.getZTreeObj Methodology随时获取
