Function(event, treeId, treeNodes, targetNode, moveType, isCopy)setting.callback.onDrop

Overview[ Dependency jquery.ztree.exedit Extension js ]

For captureNodesDragOperationEnd的EventsRevert function

IfSettings了 setting.callback.beforeDrop Methodology，且Back false，将无法Trigger onDrop EventsRevert function。

Default value：null

Function ParametersAnnotations

eventjs event Object

Standard的 js event Object

treeIdString

ObjectiveNodes targetNode Location zTree 的 treeId，便于User操控

treeNodesArray(JSON)

被Drag的Nodes JSON DataGather

IfDragOperation为 Move，treeNodes 是当前被DragNodes的DataGather。

IfDragOperation为 Copy，treeNodes 是Copy后 clone 得到的新NodesData。

targetNodeJSON

To become treeNodes DragEnd的ObjectiveNodes JSON DataObject。

IfDragTo become根Nodes，则 targetNode = null

moveTypeString

指定Move到ObjectiveNodes的相对位置

"inner"：To become子Nodes，"prev"：To become同级前一个Nodes，"next"：To become同级后一个Nodes

If moveType = null，表明Drag无效

isCopyBoolean

DragNodesOperation是 Copy 或 Move

true：Copy；false：Move

setting & function Example

1. 每次DragOperationEnd后， 弹出该被DragNodes的个数andObjectiveNodes的 tId、name Information

function zTreeOnDrop(event, treeId, treeNodes, targetNode, moveType) {
    alert(treeNodes.length + "," + (targetNode ? (targetNode.tId + ", " + targetNode.name) : "isRoot" ));
};
var setting = {
	callback: {
		onDrop: zTreeOnDrop
	}
};
