Function(treeId, treeNode)setting.callback.beforeExpand

Overview[ Dependency jquery.ztree.core Core js ]

For capture父NodesExpand之前的EventsRevert function，并且Based onBack值Sure是否AllowExpandOperation

Default value：null

Function Parameters说明

treeIdString

Correspond zTree 的 treeId，便于User操控

treeNodeJSON

要Expand的父Nodes JSON DataObject

Back值Boolean

Back值是 true / false

IfBack false，zTree 将不会ExpandNodes，也无法Trigger onExpand EventsRevert function

setting & function Example

1. BanAll已Collapse的父NodesExpand

function zTreeBeforeExpand(treeId, treeNode) {
    return false;
};
var setting = {
	callback: {
		beforeExpand: zTreeBeforeExpand
	}
};
