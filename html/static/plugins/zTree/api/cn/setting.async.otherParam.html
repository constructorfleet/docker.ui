Array(String) / JSON / Function(treeId, treeNode)setting.async.otherParam

Overview[ Dependency jquery.ztree.core Core js ]

Ajax Request提交的静态Parameters键值对。[setting.async.enable = true Effective on time]

Default value：[ ]

Array(String) FormatAnnotations

YeahEmpty[ ]，If有 key，It must exist value。 For example:：[key, value]

JSON FormatAnnotations

Direct JSON Format制作键值对，For example:：{ key1:value1, key2:value2 }

Function ParametersAnnotations

treeIdString

Correspond zTree 的 treeId，便于User操控

treeNodeJSON

YesStep Load子Nodes的的父Nodes JSON DataObject

针对根ConductStep Load时，treeNode = null

Back值Array(String) || JSON

Back值同 Array(String) || JSON Format的Data

setting Example

1. Settings Array(String) Format的Parameters

var setting = {
	async: {
		enable: true,
		url: "http://host/getNode.php",
		otherParam: ["id", "1", "name", "test"]
	}
};
ConductStep Load时，将提交Parameters id=1&name=test

2. Settings JSON Format的Parameters

var setting = {
	async: {
		enable: true,
		url: "http://host/getNode.php",
		otherParam: { "id":"1", "name":"test"}
	}
};
ConductStep Load时，将提交Parameters id=1&name=test
