String / Function(treeId, treeNode)setting.async.url

Overview[ Dependency jquery.ztree.core Core js ]

Ajax AccessData的 URL Address。[setting.async.enable = true Effective on time]

Default value：""

String Format说明

Settings固定的Step Load url Character串，请AttentionAddress的Path，确保Page能正常加载

url 内也可以带Parameters，这些Parameters就只能是Pass get Modalities提交了，并且请Attention进行转码

Function Parameters说明

treeIdString

Correspond zTree 的 treeId，便于User操控

treeNodeJSON

需要Step Load子Nodes的的父Nodes JSON DataObject

针对根进行Step Load时，treeNode = null

Back值String

Back值同 String Format的Data

setting & function Example

1. Settings异步AccessNodes的 URL 为 nodes.php

var setting = {
	async: {
		enable: true,
		url: "nodes.php",
		autoParam: ["id", "name"]
	}
};
......

2. Settings异步AccessNodes的 URL 为 function 动态Access

function getAsyncUrl(treeId, treeNode) {
    return treeNode.isParent ? "nodes1.php" : "nodes2.php";
};
var setting = {
	async: {
		enable: true,
		url: getAsyncUrl,
		autoParam: ["id", "name"]
	}
};
......
