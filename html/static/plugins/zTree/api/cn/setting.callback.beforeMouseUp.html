Function(treeId, treeNode)setting.callback.beforeMouseUp

Overview[ Dependency jquery.ztree.core Core js ]

For capture zTree 上Mouse按键松开之前的EventsRevert function，并且Based onBack值SureTrigger onMouseUp EventsRevert function

Default value：null

Function ParametersAnnotations

treeIdString

Correspond zTree 的 treeId，便于User操控

treeNodeJSON

Mouse按键松开时LocationNodes的 JSON DataObject

If不在Nodes上，则Back null

Back值Boolean

Back值是 true / false

IfBack false，将Just无法Trigger onMouseUp EventsRevert function，对OtherOperation无任何影响

setting & function Example

1. Ban onMouseUp Events

function zTreeBeforeMouseUp(treeId, treeNode) {
    return false;
};
var setting = {
	callback: {
		beforeMouseUp: zTreeBeforeMouseUp
	}
};
