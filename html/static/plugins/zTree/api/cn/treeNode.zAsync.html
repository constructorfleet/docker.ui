BooleantreeNode.zAsync

Overview[ Dependency jquery.ztree.core Core js ]

Records treeNode Nodes是否已经进行过Step Load，避免父Nodes反复Step LoadData。

InitializeNodesData时，由 zTree Increase此Properties，Don\'t advance the value

Default value：false （无子Nodes的父Nodes）； true （有子Nodes的父Nodes & LeafNodes）

Boolean Format说明

true Organisation父NodesExpand时No needAutoStep Load

false Organisation父NodesExpand时需要AutoStep Load

此ParametersIt won\'t be right reAsyncChildNodes MethodologyAny impact

treeNode Example

1. View当前被选MediumNodes是否Nodes是否需要AutoStep Load

var treeObj = $.fn.zTree.getZTreeObj("tree");
var sNodes = treeObj.getSelectedNodes();
if (sNodes.length > 0) {
	var zAsync = sNodes[0].zAsync;
}
