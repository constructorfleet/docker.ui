Function(treeNode, expandFlag, sonSign, focus, callbackFlag)zTreeObj.expandNode

Overview[ Dependency jquery.ztree.core Core js ]

Expand / Collapse AssignedNodes

v3.x 中执行此MethodologyYeahTrigger beforeExpand / onExpand 或 beforeCollapse / onCollapse EventsRevert function。便于减少冗余Code

请Pass zTree Object执行此Methodology。

Function ParametersAnnotations

treeNodeJSON

Yes Expand / Collapse 的NodesData

请务必Promise此NodesDataObject 是 zTree 内部的DataObject

expandFlagBoolean

expandFlag = true Organisation Expand Nodes

expandFlag = false Organisation Collapse Nodes

Ignore此Parameters，则Based on对此Nodes的ExpandStatusConduct toggle Switch

sonSignBoolean

sonSign = true Organisation All子孙Nodes Conduct与 expandFlag 相同的Operation

sonSign = false Organisation 只影响此Nodes，Yeah其 子孙Nodes无任何影响

sonSign = false 且 treeNode.open = expandFlag 时，不会TriggerRevert function，Direct Return

Ignore此Parameters，Equals false

focusBoolean

focus = true Organisation Expand / Collapse Operation后，PassSettings焦点Promise此焦点Enter可视区域内

focus = false Organisation Expand / Collapse Operation后，不Settings任何焦点

Ignore此Parameters，Equals true

callbackFlagBoolean

callbackFlag = true Organisation执行此Methodology时Trigger beforeExpand / onExpand 或 beforeCollapse / onCollapse EventsRevert function

callbackFlag = false Organisation执行此Methodology时Do not triggerEventsRevert function

Ignore此Parameters，Equals false

Back值Boolean

Back值Organisation最终实际Operation情况

true Organisation Expand Nodes

false Organisation Collapse Nodes

null Organisation 不是父Nodes

function Example

1. Expand当前Selection的第一个Nodes（包括其All子Nodes）

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getSelectedNodes();
if (nodes.length>0) {
	treeObj.expandNode(nodes[0], true, true, true);
}
