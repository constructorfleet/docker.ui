Function()zTreeObj.getChangeCheckedNodes

Overview[ Dependency jquery.ztree.excheck Extension js ]

Access输入框CheckStatus被改变的NodesGather（与原始Data checkedOld Comparison）。[setting.check.enable = true It works]

请Pass zTree Object执行此Methodology。

Function Parameters说明

Back值Array(JSON)

Back全部CheckStatus被改变的NodesGather Array

If需要Access每次Operation后全部被改变CheckStatus的NodesData，请在每次CheckOperation后，遍历All被改变CheckStatus的NodesData，Let it checkedOld = checked That\'s it。

function Example

1. Access当前CheckStatus被改变的NodesGather

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getChangeCheckedNodes();
