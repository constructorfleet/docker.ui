Booleansetting.data.keep.parent

Overview[ Dependency jquery.ztree.core Core js ]

zTree 的Nodes父NodesProperties锁，是否始终Hold isParent = true

Default value：false

Boolean FormatAnnotations

true / false Separately Lock / 不Lock 父NodesProperties

IfSettings为 true，则All isParent = true 的Nodes，即使该Nodes的子Nodes被AllDelete或移走，依旧Hold父NodesStatus。

setting Example

var setting = {
	data: {
		keep: {
			parent: true
		}
	}
};
