Stringsetting.data.key.url

Overview[ Dependency jquery.ztree.core Core js ]

zTree NodesDataSaveNodesLink的Objective URL 的Properties名称。

Special purpose：当后台Data只能Generate url Properties，又不想Achieved点击Nodes跳转的Functions时，可以直接Modify此Properties为Other不存在的Properties名称

Default value："url"

setting Example

1. Settings zTree ShowNodes时，将 treeNode 的 xUrl Properties当做NodesLink的Objective URL

var setting = {
	data: {
		key: {
			url: "xUrl"
		}
	}
};
