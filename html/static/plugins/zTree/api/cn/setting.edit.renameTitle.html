String / Function(treeId, treeNode)setting.edit.renameTitle

Overview[ Dependency jquery.ztree.exedit Extension js ]

Edit Namebutton Title 辅助Information。[setting.edit.enable = true & setting.edit.showRenameBtn = true Effective on time]

Default value："rename"

String Format说明

SettingsMouseMove到 Edit Namebutton Up，BrowserAuto弹出的辅助InformationContents，可Based onUser需要自行Modify

Function Parameters说明

treeIdString

Correspond zTree 的 treeId，便于User操控

treeNodeJSON

需要SettingsEdit Namebutton Title Information的Nodes JSON DataObject

Back值String

Back值同 String Format的Data

setting & function Example

1. SettingsEdit Namebutton Title 辅助Information为: "EditNode Name"

var setting = {
	edit: {
		enable: true,
		showRenameBtn: true,
		renameTitle: "EditNode Name"
	}
};
......

2. Settings父NodesEdit Namebutton Title 辅助Information为: "Edit父Node Name"

function setRenameTitle(treeId, treeNode) {
	return treeNode.isParent ? "Edit父Node Name":"EditLeafNode Name";
}
var setting = {
	edit: {
		enable: true,
		showRenameBtn: true,
		renameTitle: setRenameTitle
	}
};
......
