Boolean / Function(treeId, treeNode)setting.edit.showRemoveBtn

Overview[ Dependency jquery.ztree.exedit Extension js ]

Settings是否ShowDeletebutton。[setting.edit.enable = true Effective on time]

当点击某Nodes的Deletebutton时：

1、首先Trigger setting.callback.beforeRemove Revert function，User可判定是否进行DeleteOperation。

2、If未Settings beforeRemove 或 beforeRemove Back true，则DeleteNodes并Trigger setting.callback.onRemove Revert function。

Default value：true

Boolean Format说明

true / false Separately Show / Hide Deletebutton

Function Parameters说明

treeIdString

Correspond zTree 的 treeId，便于User操控

treeNodeJSON

需要Settings是否ShowDeletebuttonNodes JSON DataObject

Back值Boolean

Back值同 Boolean Format的Data

setting & function Example

1. Do Not ShowDeletebutton

var setting = {
	edit: {
		enable: true,
		showRemoveBtn: false
	}
};
......

2. SettingsAll的父NodesDo Not ShowDeletebutton

function setRemoveBtn(treeId, treeNode) {
	return !treeNode.isParent;
}
var setting = {
	edit: {
		enable: true,
		showRemoveBtn: setRemoveBtn
	}
};
......
