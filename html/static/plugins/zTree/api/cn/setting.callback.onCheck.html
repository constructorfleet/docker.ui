Function(event, treeId, treeNode)setting.callback.onCheck

Overview[ Dependency jquery.ztree.excheck Extension js ]

For capture checkbox / radio 被Check 或 CancelCheck的EventsRevert function

IfSettings了 setting.callback.beforeCheck Methodology，且Back false，将无法Trigger onCheck EventsRevert function。

Default value：null

Function Parameters说明

eventjs event Object

Standard的 js event Object

treeIdString

Correspond zTree 的 treeId，便于User操控

treeNodeJSON

被Check 或 CancelCheck的Nodes JSON DataObject

setting & function Example

1. Every click checkbox 或 radio 后， 弹出该Nodes的 tId、name and当前CheckStatusInformation

function zTreeOnCheck(event, treeId, treeNode) {
    alert(treeNode.tId + ", " + treeNode.name + "," + treeNode.checked);
};
var setting = {
	callback: {
		onCheck: zTreeOnCheck
	}
};
