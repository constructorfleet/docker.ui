setting.callback.onAsyncError

Function(event, treeId, treeNode, XMLHttpRequest, textStatus, errorThrown) 

Overview[ Dependency jquery.ztree.core Core js ]

For captureStep Load出现异常错误的EventsRevert function

IfSettings了 setting.callback.beforeAsync Methodology，且Back false，将无法Trigger onAsyncSuccess / onAsyncError EventsRevert function。

Default value：null

Function Parameters说明

eventjs event Object

Standard的 js event Object

treeIdString

Correspond zTree 的 treeId，便于User操控

treeNodeJSON

进行Step Load的父Nodes JSON DataObject

针对根进行Step Load时，treeNode = null

XMLHttpRequestString

Standard XMLHttpRequest Object，Please JQuery API Document。

textStatusString

RequestStatus：success，error，Please JQuery API Document。

errorThrownString

errorThrown Only if it happens，Please JQuery API Document。

setting & function Example

1. Step Load出现异常后，弹出错误Information

function zTreeOnAsyncError(event, treeId, treeNode, XMLHttpRequest, textStatus, errorThrown) {
    alert(XMLHttpRequest);
};
var setting = {
	callback: {
		onAsyncError: zTreeOnAsyncError
	}
};
