BooleantreeNode.isAjaxing

Overview[ Dependency jquery.ztree.core Core js ]

Records treeNode Nodes是否正在进行Step Load。

InitializeNodesData时，由 zTree Increase此Properties，Don\'t advance the value

Boolean Format说明

true OrganisationNodes正在进行Step Load

false OrganisationNodesNothing进行Step Load

treeNode Example

1. View当前被选MediumNodes是否Nodes正在进行Step Load

var treeObj = $.fn.zTree.getZTreeObj("tree");
var sNodes = treeObj.getSelectedNodes();
if (sNodes.length > 0) {
	var isAjaxing = sNodes[0].isAjaxing;
}
