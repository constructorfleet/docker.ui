Stringsetting.async.contentType

Overview[ Dependency jquery.ztree.core Core js ]

Ajax 提交Parameters的DataType。[setting.async.enable = true Effective on time]

Default value："application/x-www-form-urlencoded"

String Format说明

contentType = "application/x-www-form-urlencoded" I\'m satisfied绝大PartRequest，According toStandard的 Form Format提交Parameters

contentType = "application/json" I\'m satisfied .Net The programming needs，According to JSON Format提交Parameters

setting Example

1. Settings Ajax 提交Parameters的DataType为 JSON Format

var setting = {
	async: {
		enable: true,
		contentType: "application/json",
		url: "http://host/getNode.php",
		autoParam: ["id", "name"]
	}
};
