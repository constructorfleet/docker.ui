Booleansetting.async.enable

Overview[ Dependency jquery.ztree.core Core js ]

Settings zTree 是否OpenStep Load模式

Default value：false

Boolean Format说明

true Organisation Open Step Load模式

false Organisation Close Step Load模式

IfSettings为 true，请务必Settings setting.async 内的其它Parameters。

If需要根Nodes也Step Load，Initialize时 treeNodes ParametersSettings为 null Yeah。

setting Example

1. 需要OpenStep Load模式

var setting = {
	async: {
		enable: true,
		url: "http://host/getNode.php",
		autoParam: ["id", "name"]
	}
};
