JSON / Function(treeId, treeNode)setting.view.fontCss

Overview[ Dependency jquery.ztree.core Core js ]

个性化文字Styles，Only for zTree 在Nodes上Show的<A>Object。

Default value：{}

JSON Format说明

JSON Format为 JQuery cssMethodologyMedium JSON ObjectFormat，For example:：{color:"#ff0011", background:"blue"}

Function Parameters说明

treeIdString

Correspond zTree 的 treeId，便于User操控

treeNodeJSON

需要Settings自定义Styles的Nodes JSON DataObject

Back值JSON

Back值同 JSON Format的Data，For example:：{color:"#ff0011", background:"blue"}

setting & function Example

1. 不ModifyCSS，Settings全部Nodes name Show为Red

var setting = {
	view: {
		fontCss : {color:"red"}
	}
};

2. Settings level=0 的Nodes name Show为Red

function setFontCss(treeId, treeNode) {
	return treeNode.level == 0 ? {color:"red"} : {};
};
var setting = {
	view: {
		fontCss: setFontCss
	}
};
