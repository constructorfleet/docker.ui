NumbertreeNode.check_Child_State

Overview[ Dependency jquery.ztree.excheck Extension js ]

用于SettingsNodes的子Nodes的 checkBox / radio 的半选Status。[setting.check.enable = true It works]

v3.x 针对NodesDataObjectProvision treeNode.getCheckStatus() MethodologyAccessStandard的半选Status

zTree 内部Use，请勿进行Initialize 或 随意Modify

Default value：true

Number Format说明

The rules are as follows:：

setting.check.checkType = "checkbox"
treeNode.check_Child_StateCheckStatus说明
-1不存在子Nodes 或 子NodesAllSettings为 nocheck = true
0无 子Nodes被Check
1Part 子Nodes被Check
2All 子Nodes被Check

setting.check.checkType = "radio"
treeNode.check_Child_StateCheckStatus说明
-1不存在子Nodes 或 子NodesAllSettings为 nocheck = true
0无 子Nodes被Check
2有 子Nodes被Check
