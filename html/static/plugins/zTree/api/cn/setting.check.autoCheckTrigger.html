Booleansetting.check.autoCheckTrigger

Overview[ Dependency jquery.ztree.excheck Extension js ]

SettingsAuto关联Check时是否Trigger beforeCheck / onCheck EventsRevert function。[setting.check.enable = true 且 setting.check.chkStyle = "checkbox" Effective on time]

1、IfSettings setting.check.chkboxType = { "Y": "", "N": "" }，将不会有任何Auto关联Check的Operation。

2、IfOpenTrigger，YeahNodes较多的树将It affects性能，因为All被联动Check的Operation都会TriggerEventsRevert function，请Based onYes决定是否Use此Functions。

Default value: false

Boolean FormatAnnotations

true / false Separately Trigger / Do not trigger EventsRevert function

setting Example

1. YesTriggerAuto关联CheckOperation

var setting = {
	check: {
		enable: true,
		autoCheckTrigger: true
	}
};
