Function(treeId)$.fn.zTree.destroy

Overview[ Dependency jquery.ztree.core Core js ]

从 zTree v3.4 StartProvisionDestruction zTree 的Methodology。

1、用此Methodology可以Destruction指定 treeId 的 zTree，也可以DestructionCurrent PageAll的 zTree。

2、Destruction指定 treeId 的 zTree 也可以Use zTreeObj.destroy() Methodology。

3、重新Use已经被Destruction的树，必须要Use init Methodology进行Initialize。

Function Parameters说明

treeIdString

zTree 的 DOM Containers的 id

Ignore treeId，OrganisationDestructionCurrent PageAll的 zTree

Back值无

目前无任何Back值

function Example

1. Destruction id 为 "treeDemo" 的 zTree

$.fn.zTree.destroy("treeDemo");

2. DestructionAll 的 zTree

$.fn.zTree.destroy();
