Stringsetting.check.chkStyle

Overview[ Dependency jquery.ztree.excheck Extension js ]

Check框Type(checkbox 或 radio）[setting.check.enable = true Effective on time]

Default value："checkbox"

String Format说明

chkStyle = "checkbox" 时，Show checkbox Selection框，setting.check.chkboxType Properties有效。
chkStyle = "radio" 时，Show radio Selection框， setting.check.radioType Properties有效。

请Attention大小写，Don\'t change

checkbox Status说明

未Check；If是父Nodes，则无子Nodes被Check。Mouse移到该Nodes上Show为：

未Check；（只有父Nodes存在此Status）存在被Check的子Nodes。Mouse移到该Nodes上Show为：

被Check；If是父Nodes，则全部子Nodes都被Check。Mouse移到该Nodes上Show为：

被Check；（只有父Nodes存在此Status）且Part或无子Nodes被Check。Mouse移到该Nodes上Show为：

radio Status说明

未Check；If是父Nodes，则Nothing子Nodes被Check。Mouse移到该Nodes上Show为：

未Check；（只有父Nodes存在此Status）且存在被Check的子Nodes。Mouse移到该Nodes上Show为：

被Check；If是父Nodes，则Nothing子Nodes被Check。Mouse移到该Nodes上Show为：

被Check；（只有父Nodes存在此Status）且存在被Check的子Nodes。Mouse移到该Nodes上Show为：

var setting = {
