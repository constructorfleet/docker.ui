Stringsetting.data.simpleData.pIdKey

Overview[ Dependency jquery.ztree.core Core js ]

NodesData中Save其父Nodes唯一标识的Properties名称。[setting.data.simpleData.enable = true Effective on time]

Default value："pId"

setting Example

1. UseSimple Array Format的Data

var setting = {
	data: {
		simpleData: {
			enable: true,
			idKey: "id",
			pIdKey: "pId",
			rootPId: 0
		}
	}
};
var treeNodes = [
    {"id":1, "pId":0, "name":"test1"},
    {"id":11, "pId":1, "name":"test11"},
    {"id":12, "pId":1, "name":"test12"},
    {"id":111, "pId":11, "name":"test111"}
