Booleansetting.data.keep.leaf

Overview[ Dependency jquery.ztree.core Core js ]

zTree 的NodesLeafNodesProperties锁，是否始终Hold isParent = false

Default value：false

Boolean Format说明

true / false Separately Lock / 不Lock LeafNodesProperties

IfSettings为 true，则All isParent = false 的Nodes，都无法Add子Nodes。

setting Example

1. 需要LockLeafNodes状态

var setting = {
	data: {
		keep: {
			leaf: true
		}
	}
};
