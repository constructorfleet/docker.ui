Function(treeNode, addFlag, isSilent)zTreeObj.selectNode

Overview[ Dependency jquery.ztree.core Core js ]

选中指定Nodes

v3.x Support同时选中MultipleNodes。

请Pass zTree Object执行此Methodology。

Function Parameters说明

treeNodeJSON

需要被选MediumNodesData

请务必Promise此NodesDataObject 是 zTree 内部的DataObject

addFlagBoolean

addFlag = true OrganisationAppend Selection，会出现多点同时被选Medium情况

addFlag = false （Default）Organisation单独选中，原先被选MediumNodes会被UncheckStatus

setting.view.selectedMulti = false 时，此Parameters无效，Always select separately

isSilentBoolean

isSilent = true 选中Nodes时，I won\'tNodesAuto滚到到可视区域内

isSilent = false （Default）Organisation选中Nodes时，会让NodesAuto滚到到可视区域内

(v3.5.23+)

Back值无

目前无任何Back值

function Example

1. 单独选中根Nodes中第一个Nodes

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getNodes();
if (nodes.length>0) {
	treeObj.selectNode(nodes[0]);
}
