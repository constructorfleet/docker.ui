Function(parentNode, [index], newNodes, isSilent)zTreeObj.addNodes

Overview[ Dependency jquery.ztree.core Extension js ]

AddNodes。

v3.x 为了避免原先反复InitializeCaused byData重复问题，在 Initialize 和 AddNodes It\'s internal clone Operation。If需要AccessData在 zTree 内的Object，请Access此Methodology的Back值。

请Pass zTree Object执行此Methodology。

Function Parameters说明

parentNodeJSON

指定的父Nodes，IfIncrease根Nodes，请Settings parentNode 为 null Yeah。

请务必Promise此NodesDataObject 是 zTree 内部的DataObject

[index]Number

新Nodes插入Location（从 0 Start）

index = -1 时，Insert to Last

此Parameters可忽略

v3.5.19+

newNodesJSON / Array(JSON)

需要Increase的NodesData JSON ObjectGather，Data只需要满足 zTree 的NodesData必需的PropertiesYeah，详细Please“treeNode NodesData详解”

1、v3.x Support单独Add一个Nodes，即If只新增一个Nodes，You don\'t have to wrap it in an array

2、UseSimpleData模式，Please setting.data.simpleData 内的Properties说明

isSilentBoolean

设定IncreaseNodes后是否AutoExpand父Nodes。

isSilent = true 时，不Expand父Nodes，Other值或缺省Status都AutoExpand。

Back值Array(JSON)

Back值是 zTree 最终Add的NodesDataGather

If newNodes 是单个NodesData JSON，Back值也是将其包在 Array 内

Please remember：Back值MediumDataObject 是 newNodes 被 clone After，So it\'s definitely not equal！

function Example

1. Yeah id = "tree" 的 zTree Increase 1 个根Nodes

var treeObj = $.fn.zTree.getZTreeObj("tree");
var newNode = {name:"newNode1"};
newNode = treeObj.addNodes(null, newNode);

2. Yeah id = "tree" 的 zTree Increase 3 个根Nodes

var treeObj = $.fn.zTree.getZTreeObj("tree");
var newNodes = [{name:"newNode1"}, {name:"newNode2"}, {name:"newNode3"}];
newNodes = treeObj.addNodes(null, newNodes);
