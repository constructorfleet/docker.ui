Function(treeId, parentNode, responseData)setting.async.dataFilter

Overview[ Dependency jquery.ztree.core Core js ]

For right Ajax BackData进行预处理的函数。[setting.async.enable = true Effective on time]

Default value：null

Function Parameters说明

treeIdString

Correspond zTree 的 treeId，便于User操控

parentNodeJSON

进行Step Load的父Nodes JSON DataObject

对根进行Step Load时，parentNode = null

responseDataArray(JSON) / JSON / String

Step Load获取到的Data转换后的 Array(JSON) / JSON / String DataObject

v3.4Start Support XML DataFormat的 String

Back值Array(JSON) / JSON

Back值是 zTree Support的JSON DataStructure即可。

v3.x Support单个 JSON NodesData进行加载

setting & function Example

1. Modify异步获取到的NodesnameProperties

function ajaxDataFilter(treeId, parentNode, responseData) {
    if (responseData) {
      for(var i =0; i < responseData.length; i++) {
        responseData[i].name += "_filter";
      }
    }
    return responseData;
};
var setting = {
	async: {
		enable: true,
		url: "http://host/getNode.php",
		dataFilter: ajaxDataFilter
	}
};
