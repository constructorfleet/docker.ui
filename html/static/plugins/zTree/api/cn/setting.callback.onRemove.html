Function(event, treeId, treeNode)setting.callback.onRemove

Overview[ Dependency jquery.ztree.exedit Extension js ]

For captureDeleteNodes之后的EventsRevert function。

IfUserSettings了 beforeRemove Revert function，并Back false，将无法Trigger onRemove EventsRevert function。

Default value：null

Function Parameters说明

eventjs event Object

Standard的 js event Object

treeIdString

Correspond zTree 的 treeId，便于User操控

treeNodeJSON

将要Delete的Nodes JSON DataObject

setting & function Example

1. DeleteNodes时，弹出被Delete的Nodes的 tId and name Information

function zTreeOnRemove(event, treeId, treeNode) {
	alert(treeNode.tId + ", " + treeNode.name);
}
var setting = {
	callback: {
		onRemove: zTreeOnRemove
	}
};
