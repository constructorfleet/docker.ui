Function(parentNode)zTreeObj.removeChildNodes

Overview[ Dependency jquery.ztree.core Extension js ]

Clear某父Nodes的子Nodes。

1、Clear子Nodes后，父Nodes会Auto变为LeafNodes，如需要父NodesHold父NodesStatus，请Settings setting.data.keep.parent Properties。

2、请勿用此MethodologyClear根Nodes，If需要Clear根Nodes，直接Initialize zTree，并且Settings初始Nodes为 null Yeah。

3、此Methodology不会Trigger任何EventsRevert function。

请Pass zTree Object执行此Methodology。

Function Parameters说明

parentNodeJSON

需要Clear子Nodes的父NodesData

请务必Promise此NodesDataObject 是 zTree 内部的DataObject

Back值Array(JSON)

将该父Nodes的子NodesDataBack，If不存在则Back null

function Example

1. Clear选Medium第一个Nodes的子Nodes

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getSelectedNodes();
if (nodes && nodes.length>0) {
	treeObj.removeChildNodes(nodes[0]);
}
