BooleantreeNode.isParent

Overview[ Dependency jquery.ztree.core Core js ]

Records treeNode Nodes是否为父Nodes。

1、InitializeNodesData时，Based on treeNode.children PropertiesDecision，有子Nodes则Settings为 true，否And.. false

2、InitializeNodesData时，If设定 treeNode.isParent = true，即使无子NodesData，也会Settings为父Nodes

3、为了SolvePart朋友Generate json Data出现的兼容问题, Support "false","true" Character串Format的Data

Boolean Format说明

true Organisation是父Nodes

false Organisation不是父Nodes

var treeObj = $.fn.zTree.getZTreeObj("tree");
var sNodes = treeObj.getSelectedNodes();

	var isParent = sNodes[0].isParent;
