Boolean / Function(treeId, treeNode)setting.view.dblClickExpand

Overview[ Dependency jquery.ztree.core Core js ]

双击Nodes时，是否AutoExpand父Nodes的标识

Default value: true

Boolean Format说明

true / false Separately双击Nodes Switch / 不Switch ExpandStatus

Function Parameters说明

treeIdString

Correspond zTree 的 treeId，便于User操控

treeNodeJSON

需要Settings是否双击SwitchExpandStatus的Nodes JSON DataObject

Back值Boolean

Back值同 Boolean Format的Data
