Boolean / Function(treeId, treeNode)setting.view.showIcon

Overview[ Dependency jquery.ztree.core Core js ]

Settings zTree 是否ShowNodes的Icon。

Default value：true

Boolean Format说明

true / false Separately Show / Hide Icon

Function Parameters说明

treeIdString

Correspond zTree 的 treeId，便于User操控

treeNodeJSON

需要Settings是否ShowIcon的Nodes JSON DataObject

Back值Boolean

Back值同 Boolean Format的Data

setting & function Example

1. Settings zTree Do Not ShowIcon

var setting = {
	view: {
		showIcon: false
	}
};
......

2. Settings zTree Just level=2 的NodesDo Not ShowIcon

function showIconForTree(treeId, treeNode) {
	return treeNode.level != 2;
};
var setting = {
	view: {
		showIcon: showIconForTree
	}
};
......
