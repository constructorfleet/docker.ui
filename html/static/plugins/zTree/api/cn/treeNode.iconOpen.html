StringtreeNode.iconOpen

Overview[ Dependency jquery.ztree.core Core js ]

父Nodes自定义Expand时Icon的 URL Path。

1、此PropertiesOnly for父Nodes有效

2、此Properties必须与 iconClose 同时Use

3、If想Utilization className Settings个性化Icon，YesSettings treeNode.iconSkin Properties

Default value：无

String FormatAnnotations

IconPicture的 url It could be相对Path也It could be绝对Path

Settings相对Path请AttentionPage与Picture之间的关系，确保Picture能够正常加载

treeNode Example

1. SettingsNodes的个性化Icon

	//父NodesExpand Collapse时Use相同的Icon

	//LeafNodes个性化Icon
