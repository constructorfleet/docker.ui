Function(treeId, treeNode)setting.callback.beforeRightClick

Overview[ Dependency jquery.ztree.core Core js ]

For capture zTree 上MouseRight点击之前的EventsRevert function，并且Based onBack值SureTrigger onRightClick EventsRevert function

Default value：null

Function Parameters说明

treeIdString

Correspond zTree 的 treeId，便于User操控

treeNodeJSON

MouseRight点击时LocationNodes的 JSON DataObject

If不在Nodes上，则Back null

Back值Boolean

Back值是 true / false

IfBack false，将Just无法Trigger onRightClick EventsRevert function，对OtherOperation无任何影响

setting & function Example

1. Ban onRightClick Events

function zTreeBeforeRightClick(treeId, treeNode) {
    return false;
};
var setting = {
	callback: {
		beforeRightClick: zTreeBeforeRightClick
	}
};
