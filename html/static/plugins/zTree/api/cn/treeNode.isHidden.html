BooleantreeNode.isHidden

Overview[ Dependency jquery.ztree.exhide Extension js ]

Decision treeNode Nodes是否被Hide。

1、Initialize zTree 时，IfNodesSettings isHidden = true，会被AutoHide

2、请勿对已加载的NodesModify此Properties，Hide / Show 请Use hideNode() / hideNodes() / showNode() / showNodes() Methodology

Boolean Format说明

true Organisation被Hide

false Organisation被Show

treeNode Example

1. View第一个根Nodes是否被Hide

var treeObj = $.fn.zTree.getZTreeObj("tree");
var sNodes = treeObj.getNodes();
if (sNodes.length > 0) {
	var isHidden = sNodes[0].isHidden;
}
