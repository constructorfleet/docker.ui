Function(filter, isSingle, parentNode, invokeParam)zTreeObj.getNodesByFilter

Overview[ Dependency jquery.ztree.core Core js ]

Based on自定义规则SearchNodesData JSON ObjectGather 或 单个NodesData

可自定义复杂的Search规则

请Pass zTree Object执行此Methodology。

Function ParametersAnnotations

filterFunction

Custom Filter Functions function filter(node) {...}

filter Parameters：node (NodesData JSON)

filter Back值：boolean (true Organisation符合Search条件；false Organisation不符合Search条件)

isSingleBoolean

isSingle = true Organisation只Find单个Nodes

isSingle = false OrganisationFindNodesGather

忽略此Parameters，OrganisationFindNodesGather

parentNodeJSON

Yeah指定在某个父Nodes下的子Nodes中Search

忽略此Parameters，Organisation在AllNodes中Search

invokeParam任意Type

User自定义的DataObject，For filter 中Conduct计算

Back值Array(JSON) / JSON

isSingle = true Back 第一个找到的NodesData JSON，无结果时Back null

isSingle = false Back NodesDataGather Array(JSON)，无结果时Back [ ]

function Example

1. Find level = 2 & name Organisation "test" 的NodesData

function filter(node) {
    return (node.level == 2 && node.name.indexOf("test")>-1);
}
......
var treeObj = $.fn.zTree.getZTreeObj("tree");
var node = treeObj.getNodesByFilter(filter, true); // 仅Find一个Nodes
var nodes = treeObj.getNodesByFilter(filter); // FindNodesGather
