Function(treeNode)zTreeObj.cancelSelectedNode

Overview[ Dependency jquery.ztree.core Core js ]

CancelNodes的选中Status。

v3.x Support多点同时选中，因此UncheckStatusYeahAllCancel，也Yeah单独Cancel某个Nodes的选中Status。

请Pass zTree Object执行此Methodology。

Function Parameters说明

treeNodeJSON

需要UncheckStatus的Nodes。

请务必Promise此NodesDataObject 是 zTree 内部的DataObject

IfIgnore此Parameters，则将CancelAll被选中Nodes的选中Status。

Back值无

目前无任何Back值

function Example

1. Cancel当前All被选中Nodes的选中Status

var treeObj = $.fn.zTree.getZTreeObj("tree");
treeObj.cancelSelectedNode();

2. CancelCurrent No一个被选中Nodes的选中Status

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getSelectedNodes();
if (nodes.length>0) { 
	treeObj.cancelSelectedNode(nodes[0]);
}
