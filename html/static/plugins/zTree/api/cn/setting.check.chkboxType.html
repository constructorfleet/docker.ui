JSONsetting.check.chkboxType

Overview[ Dependency jquery.ztree.excheck Extension js ]

Check checkbox Yeah父子Nodes的关联关系。[setting.check.enable = true 且 setting.check.chkStyle = "checkbox" Effective on time]

Default value：{ "Y": "ps", "N": "ps" }

JSON Format说明

Y Properties定义 checkbox 被CheckAfter情况；
N Properties定义 checkbox CancelCheckAfter情况；
"p" OrganisationOperationIt affects父级Nodes；
"s" OrganisationOperationIt affects子级Nodes。

请Attention大小写，Don\'t change

setting Example

1. checkbox CheckOperation，只影响父级Nodes；CancelCheckOperation，只影响子级Nodes

var setting = {
	check: {
		enable: true,
		chkStyle: "checkbox",
		chkboxType: { "Y": "p", "N": "s" }
	}
};
