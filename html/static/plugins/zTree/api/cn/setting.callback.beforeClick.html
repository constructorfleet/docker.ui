Function(treeId, treeNode, clickFlag)setting.callback.beforeClick

Overview[ Dependency jquery.ztree.core Core js ]

For captureClickNodes之前的EventsRevert function，并且Based onBack值Sure是否AllowClickOperation

Default value：null

Function Parameters说明

treeIdString

Correspond zTree 的 treeId，便于User操控

treeNodeJSON

被Click的Nodes JSON DataObject

clickFlagNumber

Nodes被点击后的选中OperationType，Look at the table

Back值Boolean

Back值是 true / false

IfBack false，zTree 将不会选中Nodes，也无法Trigger onClick EventsRevert function

function zTreeBeforeClick(treeId, treeNode, clickFlag) {
    return (treeNode.id !== 1);
