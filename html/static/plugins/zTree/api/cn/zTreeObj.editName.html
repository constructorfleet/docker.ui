Function(treeNode)zTreeObj.editName

Overview[ Dependency jquery.ztree.exedit Extension js ]

Settings某NodesEnterEdit NameStatus。

1、IfYes用 js CancelEdit NameStatus，请Use cancelEditName(newName) Methodology。

2、可Utilization此Methodology让当前正Edit的Nodes input 输入框Access焦点。

3、请Pass zTree Object执行此Methodology。

Function ParametersAnnotations

treeNodeJSON

指定EnterEdit NameStatus的Nodes JSON Data

请务必Promise此NodesDataObject 是 zTree 内部的DataObject

Back值无

目前无任何Back值
