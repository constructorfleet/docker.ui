Function(targetNode, treeNode, moveType, isSilent)zTreeObj.moveNode

Overview[ Dependency jquery.ztree.exedit Extension js ]

MoveNodes。

请Pass zTree Object执行此Methodology。

Function ParametersAnnotations

targetNodeJSON

要Move到的ObjectiveNodes JSON Data

IfMove成为根Nodes，请Settings targetNode 为 null Yeah

请务必Promise此NodesDataObject 是 zTree 内部的DataObject

treeNodeJSON

需要被Move的NodesData

请务必Promise此NodesDataObject 是 zTree 内部的DataObject

moveTypeString

指定Move到ObjectiveNodes的相对位置

"inner"：成为子Nodes，"prev"：成为同级前一个Nodes，"next"：成为同级后一个Nodes

isSilentBoolean

设定MoveNodes后是否AutoExpand父Nodes。

isSilent = true 时，不Expand父Nodes，Other值或缺省Status都AutoExpand。

Back值JSON

Back值是最终被Move的NodesData，Under normal circumstances treeNode Parameters完全相同

If Back值 为 null，Annotations MoveFailed，The main reasons are:：
 1、targetNode 是 treeNode 父Nodes，且 moveType = "inner"
 2、targetNode 是 treeNode 子孙Nodes

function Example

1. 将根Nodes中第二个Nodes Move成为 第一个Nodes的子Nodes

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getNodes();
treeObj.moveNode(nodes[0], nodes[1], "inner");

2. 将根Nodes中第二个Nodes Move成为 第一个Nodes的前一个Nodes

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getNodes();
treeObj.moveNode(nodes[0], nodes[1], "prev");
