Function(treeNodes)zTreeObj.hideNodes

Overview[ Dependency jquery.ztree.exhide Extension js ]

Hide一批Nodes。

1、此Functions不Support exedit Extension，因此不要在编辑状态时UseHideNodes的Methodology。

2、Hide/ShowNodes，It affectsNodes的 isFirstNode 和 isLastNode Properties。

3、请Pass zTree Object执行此Methodology。

Function Parameters说明

treeNodesArray(JSON)

指定被Hide的Nodes JSON DataGather

请务必Promise这些NodesDataObject 是 zTree 内部的DataObject

Back值无

目前无任何Back值

function Example

1. Hide根Nodes第一个Nodes的子Nodes

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getNodes();
treeObj.hideNodes(nodes[0].children);
