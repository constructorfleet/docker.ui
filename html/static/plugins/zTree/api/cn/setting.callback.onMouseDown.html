Function(event, treeId, treeNode)setting.callback.onMouseDown

Overview[ Dependency jquery.ztree.core Core js ]

For capture zTree 上Mouse按键Press后的EventsRevert function

IfSettings了 setting.callback.beforeMouseDown Methodology，且Back false，将无法Trigger onMouseDown EventsRevert function。

Default value：null

Function Parameters说明

eventjs event Object

Standard的 js event Object

treeIdString

Correspond zTree 的 treeId，便于User操控

treeNodeJSON

Mouse按键Press时所在Nodes的 JSON DataObject

If不在Nodes上，则Back null

setting & function Example

1. 每次Mouse按键Press后， 弹出Mouse所在Nodes的 tId、name Information

function zTreeOnMouseDown(event, treeId, treeNode) {
    alert(treeNode ? treeNode.tId + ", " + treeNode.name : "isRoot");
};
var setting = {
	callback: {
		onMouseDown: zTreeOnMouseDown
	}
};
