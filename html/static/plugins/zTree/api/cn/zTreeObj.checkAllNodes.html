Function(checked)zTreeObj.checkAllNodes

Overview[ Dependency jquery.ztree.excheck Extension js ]

Check 或 CancelCheck 全部Nodes。[setting.check.enable = true 且 setting.check.chkStyle = "checkbox" It works]

此Methodology不会Trigger beforeCheck / onCheck EventsRevert function。

请Pass zTree Object执行此Methodology。

Function Parameters说明

checkedBoolean

checked = true OrganisationCheck全部Nodes

checked = false Organisation全部NodesCancelCheck

不It affects treeNode.nochecked = true 的Nodes。

不It affects未加载的Nodes。

Back值无

目前无任何Back值

function Example

1. Check全部Nodes

var treeObj = $.fn.zTree.getZTreeObj("tree");
treeObj.checkAllNodes(true);
