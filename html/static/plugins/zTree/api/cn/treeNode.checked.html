BooleantreeNode.checked

Overview[ Dependency jquery.ztree.excheck Extension js ]

Nodes的 checkBox / radio 的 CheckStatus。[setting.check.enable = true & treeNode.nocheck = false It works]

1、If不Use checked PropertiesSettingsCheckStatus，请Modify setting.data.key.checked

2、Create treeNode Data时Settings treeNode.checked = true Yeah让Nodes的输入框Default asCheckStatus

3、ModifyNodesCheckStatus，YeahUse treeObj.checkNode / checkAllNodes / updateNode Methodology，具体Use哪种请Based on自己的需求而定

4、为了SolvePart朋友Generate json Data出现的兼容问题, Support "false","true" Character串Format的Data

Default value：false

Boolean FormatAnnotations

true OrganisationNodes的输入框被Check

false OrganisationNodes的输入框未Check

treeNode Example

1. Initialize的DataSettings Default asCheckStatus

var nodes = [
{ "id":1, "name":"test1", checked:true },
{ "id":2, "name":"test2", checked:true }
]

2. Access第一个根Nodes的CheckStatus

var treeObj = $.fn.zTree.getZTreeObj("tree");
var checked = treeObj.getNodes()[0].checked;
