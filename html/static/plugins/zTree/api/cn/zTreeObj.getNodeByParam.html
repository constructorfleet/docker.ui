Function(key, value, parentNode)zTreeObj.getNodeByParam

Overview[ Dependency jquery.ztree.core Core js ]

Based onNodesData的PropertiesSearch，Access条件完全匹配的NodesData JSON Object

请Pass zTree Object执行此Methodology。

Function Parameters说明

keyString

需要精确匹配的Properties名称

value?

需要精确匹配的Properties值，It could be任何Type，只要Promise与 key AssignedProperties值Hold一致Yeah

parentNodeJSON

SearchScope，指定在某个父Nodes下的子Nodes中进行Search

忽略此Parameters，Organisation在AllNodes中Search

Back值JSON

匹配精确Search的NodesData

1、If nothing happens，Back null

2、如有MultipleNodes满足Question条件，只Back第一个匹配到的Nodes

function Example

1. Find id = 1 的NodesData

var treeObj = $.fn.zTree.getZTreeObj("tree");
var node = treeObj.getNodeByParam("id", 1, null);
