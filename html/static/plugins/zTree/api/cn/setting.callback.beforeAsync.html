Function(treeId, treeNode)setting.callback.beforeAsync

Overview[ Dependency jquery.ztree.core Core js ]

For captureStep Load之前的EventsRevert function，zTree Based onBack值Sure是否AllowConductStep Load

Default value：null

Function ParametersAnnotations

treeIdString

Correspond zTree 的 treeId，便于User操控

treeNodeJSON

ConductStep Load的父Nodes JSON DataObject

针对根ConductStep Load时，treeNode = null

Back值Boolean

Back值是 true / false

IfBack false，zTree 将不ConductStep Load，也无法Trigger onAsyncSuccess / onAsyncError EventsRevert function

setting & function Example

1. Ban id 为 1 的父NodesConductStep LoadOperation

function zTreeBeforeAsync(treeId, treeNode) {
    return (treeNode.id !== 1);
};
var setting = {
	callback: {
		beforeAsync: zTreeBeforeAsync
	}
};
