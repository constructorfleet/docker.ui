Function(treeId, treeNode)setting.callback.beforeRemove

Overview[ Dependency jquery.ztree.exedit Extension js ]

For captureNodes被Delete之前的EventsRevert function，并且Based onBack值Sure是否AllowDeleteOperation

Default value：null

Function Parameters说明

treeIdString

Correspond zTree 的 treeId，便于User操控

treeNodeJSON

将要Delete的Nodes JSON DataObject

Back值Boolean

Back值是 true / false

IfBack false，zTree 将不DeleteNodes，也无法Trigger onRemove EventsRevert function

setting & function Example

1. Ban全部DeleteOperation

function zTreeBeforeRemove(treeId, treeNode) {
	return false;
}
var setting = {
	edit: {
		enable: true
	},
	callback: {
		beforeRemove: zTreeBeforeRemove
	}
};
