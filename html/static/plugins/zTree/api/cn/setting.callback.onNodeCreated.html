Function(event, treeId, treeNode)setting.callback.onNodeCreated

Overview[ Dependency jquery.ztree.core Core js ]

用于捕获NodesGenerate DOM 后的EventsRevert function

v3.x Delayed loading technology used，因此Yeah父Nodes未Expand的子Nodes来说，Initialize后是不会Trigger此Revert function，直到其父Nodes被Expand

大Data量的Nodes加载请Attention：不Settings onNodeCreated，可以提升一PartInitialize性能

Default value：null

Function Parameters说明

eventjs event Object

Standard的 js event Object

treeIdString

Correspond zTree 的 treeId，便于User操控

treeNodeJSON

Generate DOM Over的Nodes的 JSON DataObject

setting & function Example

1. 创建Nodes DOM 后， 弹出该Nodes的 tId、name Information

function zTreeOnNodeCreated(event, treeId, treeNode) {
    alert(treeNode.tId + ", " + treeNode.name);
};
var setting = {
	callback: {
		onNodeCreated: zTreeOnNodeCreated
	}
};
