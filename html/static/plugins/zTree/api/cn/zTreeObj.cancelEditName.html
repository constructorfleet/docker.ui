Function(newName)zTreeObj.cancelEditName

Overview[ Dependency jquery.ztree.exedit Extension js ]

CancelNodes的Edit NameStatus，YeahRestore原名称，也Yeah强行赋给新的名称。

请Pass zTree Object执行此Methodology。

Function ParametersAnnotations

newNameString

Reset New Name。

IfIgnore此Parameters，则Restore原名称。

Back值无

目前无任何Back值

function Example

1. Cancel zTree 的Edit NameStatus，Restore该Nodes原有名称

var treeObj = $.fn.zTree.getZTreeObj("tree");
treeObj.cancelEditName();

2. Cancel zTree 的Edit NameStatus，并且重新设定该Node Name

var treeObj = $.fn.zTree.getZTreeObj("tree");
treeObj.cancelEditName("test_new_name");
