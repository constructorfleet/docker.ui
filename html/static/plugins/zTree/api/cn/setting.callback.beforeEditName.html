Function(treeId, treeNode)setting.callback.beforeEditName

Overview[ Dependency jquery.ztree.exedit Extension js ]

用于捕获Nodes编辑button click Events，并且Based onBack值Sure是否允许进入名称编辑状态

此EventsRevert function最主要是用于捕获编辑button点击Events，然后Trigger自定义的编辑界面Operation。

Default value：null

Function Parameters说明

treeIdString

Correspond zTree 的 treeId，便于User操控

treeNodeJSON

将要进入Edit Name状态的Nodes JSON DataObject

Back值Boolean

Back值是 true / false

IfBack false，Nodes将无法进入 zTree Default的Edit Name状态

setting & function Example

1. 禁止Modify父Nodes的名称

function zTreeBeforeEditName(treeId, treeNode) {
	return !treeNode.isParent;
}
var setting = {
	edit: {
		enable: true
	},
	callback: {
		beforeEditName: zTreeBeforeEditName
	}
};
