Function(event, treeId, treeNode)setting.callback.onExpand

Overview[ Dependency jquery.ztree.core Core js ]

For captureNodes被Expand的EventsRevert function

IfSettings了 setting.callback.beforeExpand Methodology，且Back false，将无法Trigger onExpand EventsRevert function。

Default value：null

Function Parameters说明

eventjs event Object

Standard的 js event Object

treeIdString

Correspond zTree 的 treeId，便于User操控

treeNodeJSON

被Expand的Nodes JSON DataObject

setting & function Example

1. 每次ExpandNodes后， 弹出该Nodes的 tId、name Information

function zTreeOnExpand(event, treeId, treeNode) {
    alert(treeNode.tId + ", " + treeNode.name);
};
var setting = {
	callback: {
		onExpand: zTreeOnExpand
	}
};
