Function()zTreeObj.getNodes

Overview[ Dependency jquery.ztree.core Core js ]

Access zTree 的全部NodesData

请Pass zTree Object执行此Methodology。

Function Parameters说明

Back值Array(JSON)

全部NodesData

1、Array Just是根Nodes的Gather（Default情况子Nodes都处于 children Properties下）；

2、如需遍历全部Nodes需要Utilization递归，或Utilization transformToArray Methodology 将Data变成Simple的 Array Gather

3、YeahStep LoadMode，尚未加载的子Nodes是无法Pass此MethodologyAccess的。

function Example

1. Access全部NodesData

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getNodes();
