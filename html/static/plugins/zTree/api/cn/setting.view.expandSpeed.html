String / Numbersetting.view.expandSpeed

Overview[ Dependency jquery.ztree.core Core js ]

zTree NodesExpand、Collapse时的动画速度，SettingsMethodology同 JQuery Animating effects speed Parameters。

IE6 下会AutoClose动画效果，以Promise zTree 的Operation速度

Default value："fast"

String FormatAnnotations

三种预定速度之一的Character串("slow", "normal", or "fast")

Settings为 "" 时，Do Not Show动画效果

Number FormatAnnotations

Organisation动画时长的毫秒数值 (如：1000)

setting Example

1. Settings为慢速Show动画效果

var setting = {
	view: {
		expandSpeed: "slow"
	}
};
