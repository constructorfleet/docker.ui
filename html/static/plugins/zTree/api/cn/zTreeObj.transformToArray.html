Function(treeNodes)zTreeObj.transformToArray

Overview[ Dependency jquery.ztree.core Core js ]

将 zTree Use的Standard JSON 嵌套Format的Data转换为Simple Array Format。(免去User自行编写递归遍历全部Nodes的麻烦)

请Pass zTree Object执行此Methodology。

Function Parameters说明

treeNodesArray(JSON) / JSON

To be converted zTree NodesDataObjectGather 或 某个单独Nodes的DataObject

Back值Array(JSON)

转换后的Simple Array DataFormat

function Example

1. 将 zTree NodesData转换为Simple Array Format

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.transformToArray(treeObj.getNodes());
