Function(treeNodes)zTreeObj.showNodes

Overview[ Dependency jquery.ztree.exhide Extension js ]

Show一批已经被Hide的Nodes。

1、此Functions不Support exedit Extension，因此不要在EditStatus时UseHideNodes的Methodology。

2、Hide/ShowNodes，It affectsNodes的 isFirstNode 和 isLastNode Properties。

3、请Pass zTree Object执行此Methodology。

Function ParametersAnnotations

treeNodesArray(JSON)

指定被Show的Nodes JSON DataGather

请务必Promise这些NodesDataObject 是 zTree 内部的DataObject

Back值无

目前无任何Back值

function Example

1. ShowAllHide的Nodes

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getNodesByParam("isHidden", true);
treeObj.showNodes(nodes);
