Numbersetting.edit.drag.minMoveSize

Overview[ Dependency jquery.ztree.exedit Extension js ]

判定是否DragOperation的最小位移值 (Units：px)。[setting.edit.enable = true Effective on time]

Based on自己的需求可适当调整此值，If太小容易导致点击Mouse时误OperationConductDrag

Default value：5

setting Example

1. ChangeDragOperation启动位移值为10px

var setting = {
	edit: {
		enable: true,
		drag: {
			minMoveSize: 10
		}
	}
};
