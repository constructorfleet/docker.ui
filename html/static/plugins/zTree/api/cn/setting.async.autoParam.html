Array(String) / Function(treeId, treeNode)setting.async.autoParam

Overview[ Dependency jquery.ztree.core Core js ]

Step Load时YesAuto提交父NodesProperties的Parameters。[setting.async.enable = true Effective on time]

Default value：[ ]

Array(String) FormatAnnotations

1、将Yes作为Parameters提交的Properties名称，Made Array Yeah，For example:：["id", "name"]

2、YeahSettings提交时的Parameters名称，For example: server 只Accept zId : ["id=zId"]

Function ParametersAnnotations

treeIdString

Correspond zTree 的 treeId，便于User操控

treeNodeJSON

YesStep Load子Nodes的的父Nodes JSON DataObject

针对根ConductStep Load时，treeNode = null

Back值Array(String)

Back值同 Array(String) Format的Data

setting Example

1. Settings id Properties为Auto提交的Parameters

var setting = {
	async: {
		enable: true,
		url: "http://host/getNode.php",
		autoParam: ["id"]
	}
};
Assumptions Step Load 父Nodes(node = {id:1, name:"test"}) 的子Nodes时，将提交Parameters id=1
......

2. Settings id Properties作为 zId To becomeAuto提交的Parameters

var setting = {
	async: {
		enable: true,
		url: "http://host/getNode.php",
		autoParam: ["id=zId"]
	}
};
Assumptions 对父Nodes node = {id:1, name:"test"}，ConductStep Load时，将提交Parameters zId=1
......
