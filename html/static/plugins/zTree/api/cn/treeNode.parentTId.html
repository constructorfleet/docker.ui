StringtreeNode.parentTId

Overview[ Dependency jquery.ztree.core Core js ]

treeNode Nodes的父Nodes唯一标识 tId。

1、v3.x 用 parentTId Replaced the original parentNode Properties，同时Increase了 getParentNode Methodology，To avoid the original parentNode Caused by clone Dead circulation

2、InitializeNodesData时，由 zTree Increase此Properties，Don\'t advance the value

String Format说明

zTree 内部生成的Nodes唯一标识，Please treeNode.tId Note

If treeNode 是根Nodes，则 parentTId = null

treeNode Example

1. View当前被选MediumNodes的父Nodes tId

var treeObj = $.fn.zTree.getZTreeObj("tree");
var sNodes = treeObj.getSelectedNodes();
if (sNodes.length > 0) {
	var parentTId = sNodes[0].parentTId;
}
