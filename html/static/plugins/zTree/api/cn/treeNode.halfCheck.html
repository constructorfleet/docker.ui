BooleantreeNode.halfCheck

Overview[ Dependency jquery.ztree.excheck Extension js ]

强制Nodes的 checkBox / radio 的 半CheckStatus。[setting.check.enable = true & treeNode.nocheck = false It works]

1、强制为半CheckStatus后，不再进行Auto计算半CheckStatus

2、Settings treeNode.halfCheck = false 或 null 才能RestoreAuto计算半CheckStatus

3、为了SolvePart朋友Generate json Data出现的兼容问题, Support "false","true" Character串Format的Data

Default value：false

Boolean Format说明

true OrganisationNodes的输入框 强行Settings为半Check

false OrganisationNodes的输入框 Based on zTree 的规则Auto计算半CheckStatus

treeNode Example

1. Initialize的DataSettings Default as半CheckStatus

var nodes = [
{ "id":1, "name":"test1", isParent:true, checked:true, halfCheck:true },

{ "id":3, "name":"test3", isParent:true, checked:true },
{ "id":4, "name":"test4", isParent:true, checked:false }
