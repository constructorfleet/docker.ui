Function(expandFlag)zTreeObj.expandAll

Overview[ Dependency jquery.ztree.core Core js ]

Expand / Collapse 全部Nodes

此Methodology不会Trigger beforeExpand / onExpand 和 beforeCollapse / onCollapse EventsRevert function。

请Pass zTree Object执行此Methodology。

Function Parameters说明

expandFlagBoolean

expandFlag = true Organisation Expand 全部Nodes

expandFlag = false Organisation Collapse 全部Nodes

Back值Boolean

Back值Organisation最终实际Operation情况

true Organisation Expand 全部Nodes

false Organisation Collapse 全部Nodes

null Organisation 不存在任何父Nodes

function Example

1. Expand全部Nodes

var treeObj = $.fn.zTree.getZTreeObj("tree");
treeObj.expandAll(true);
