Function(key, value, parentNode)zTreeObj.getNodesByParam

Overview[ Dependency jquery.ztree.core Core js ]

Based onNodesData的PropertiesSearch，Access条件完全匹配的NodesData JSON ObjectGather

请Pass zTree Object执行此Methodology。

Function ParametersAnnotations

keyString

需要精确匹配的Properties名称

value?

需要精确匹配的Properties值，It could be任何Type，只要Promise与 key AssignedProperties值Hold一致Yeah

parentNodeJSON

Yeah指定在某个父Nodes下的子Nodes中Search

忽略此Parameters，Organisation在AllNodes中Search

Back值Array(JSON)

匹配精确Search的NodesDataGather

If nothing happens，Back [ ]
