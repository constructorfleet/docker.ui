Boolean / Function(treeId, treeNode)setting.view.showTitle

Overview[ Dependency jquery.ztree.core Core js ]

Settings zTree 是否ShowNodes的 title HintInformation(即Nodes DOM 的 title Properties)。

Please be with setting.data.key.title 同时Use。

Default value：true

Boolean Format说明

true / false Separately Show / Hide HintInformation

If setting.view.showTitle = true & setting.data.key.title = ''，zTree 会AutoUse setting.data.key.name 指定的Node Name当做 title

Function Parameters说明

treeIdString

Correspond zTree 的 treeId，便于User操控

treeNodeJSON

需要Settings是否ShowHintInformation的Nodes JSON DataObject

Back值Boolean

Back值同 Boolean Format的Data

setting & function Example

1. Settings zTree Do Not ShowHintInformation

var setting = {
	view: {
		showTitle: false
	}
};
......

2. Settings zTree Just level=2 的NodesDo Not ShowHintInformation

function showTitleForTree(treeId, treeNode) {
	return treeNode.level != 2;
};
var setting = {
	view: {
		showTitle: showTitleForTree
	}
};
......
