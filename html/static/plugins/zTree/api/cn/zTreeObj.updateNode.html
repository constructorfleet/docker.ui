Function(treeNode, checkTypeFlag)zTreeObj.updateNode

Overview[ Dependency jquery.ztree.core Core js ]

Update某NodesData，Mainly for该NodesShowProperties的Update。

1、Available name、target、 url、icon、 iconSkin、checked、nocheck 等这几个用于Show效果的Parameters进行Update，Other用于 zTreeNodes 的Parameters请不要随意Update，YeahExpandNodes，Please call expandNodeMethodology，因此请勿随意Modify open Properties。

2、用此MethodologyModify checked CheckStatus不会Trigger beforeCheck / onCheck EventsRevert function。

请Pass zTree Object执行此Methodology。

Function Parameters说明

treeNodeJSON

指定需要Update的Nodes JSON Data

请务必Promise此NodesDataObject 是 zTree 内部的DataObject

checkTypeFlagBoolean

checkTypeFlag = true OrganisationAccording to setting.check.chkboxType Properties进行父子Nodes的Check联动Operation

checkTypeFlag = false Organisation只Modify此NodesCheckStatus，无任何Check联动Operation

当 setting.check.enable = true 且 setting.check.chkStyle = "checkbox" It\'ll work

不Influence father and sonNodes中 treeNode.nochecked = true 的Nodes。

Back值无

目前无任何Back值

function Example

1. Update根Nodes中第一个Nodes的名称

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getNodes();
if (nodes.length>0) {
	nodes[0].name = "test";
	treeObj.updateNode(nodes[0]);
}
