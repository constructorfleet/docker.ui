Function(treeId, treeNode)setting.callback.beforeCheck

Overview[ Dependency jquery.ztree.excheck Extension js ]

For capture Check 或 CancelCheck 之前的EventsRevert function，并且Based onBack值Sure是否Allow Check 或 CancelCheck

Default value：null

Function Parameters说明

treeIdString

Correspond zTree 的 treeId，便于User操控

treeNodeJSON

Conduct Check 或 CancelCheck 的Nodes JSON DataObject

Back值Boolean

Back值是 true / false

IfBack false，将不会改变CheckStatus，并且无法Trigger onCheck EventsRevert function

setting & function Example

1. BanAllCheckOperation，HoldInitialize的CheckStatus

function zTreeBeforeCheck(treeId, treeNode) {
    return false;
};
var setting = {
	callback: {
		beforeCheck: zTreeBeforeCheck
	}
};
