Function(treeId, treeNode)setting.callback.beforeDragOpen

Overview[ Dependency jquery.ztree.exedit Extension js ]

For captureDragNodesMove到CollapseStatus的父Nodes后，即将AutoExpand该父Nodes之前的EventsRevert function，并且Based onBack值Sure是否AllowAutoExpandOperation

Default value：null

Function Parameters说明

treeIdString

需要被Expand的父Nodes treeNode Location zTree 的 treeId，便于User操控

treeNodeJSON

要被AutoExpand的父Nodes JSON DataObject

Back值Boolean

Back值是 true / false

IfBack false，zTree 将无法进行AutoExpandOperation

setting & function Example

1. Ban全部Drag时的AutoExpandOperation

function zTreeBeforeDragOpen(treeId, treeNode) {
    return false;
};
var setting = {
	edit: {
		enable: true
	},
	callback: {
		beforeDragOpen: zTreeBeforeDragOpen
	}
};
