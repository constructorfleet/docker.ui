Function(treeNode)zTreeObj.getNodeIndex

Overview[ Dependency jquery.ztree.core Core js ]

Access某Nodes在同级NodesMedium序号（从0Start）

请Pass zTree Object执行此Methodology。

Function Parameters说明

treeNodeJSON

需要Question顺序的Nodes JSON DataObject

请务必Promise此NodesDataObject 是 zTree 内部的DataObject

Back值Number

Back值从 0 Start计数

If不存在该NodesData，Back -1

function Example

1. Access当前选Medium第一个Nodes在同级NodesMedium序号

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getSelectedNodes();
if (nodes.length>0) {
	var index = treeObj.getNodeIndex(nodes[0]);
}
