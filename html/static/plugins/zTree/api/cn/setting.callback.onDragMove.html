Function(event, treeId, treeNodes)setting.callback.onDragMove

Overview[ Dependency jquery.ztree.exedit Extension js ]

For captureNodes被Drag过程中Move的EventsRevert function

Mainly for捕获 zTree NodesDrag到的 DOM，从而OperationCorrespond的 DOM。

Default value：null

eventjs event Object

Standard的 js event Object

treeIdString

被Drag的Nodes treeNodes Location zTree 的 treeId，便于User操控

treeNodesArray(JSON)

要被Drag的Nodes JSON DataGather
