Numbersetting.edit.drag.maxShowNodeNum

Overview[ Dependency jquery.ztree.exedit Extension js ]

DragMultiple兄弟Nodes时，浮动图层中Show的最大Nodes数。 多余的Nodes用...Replace。[setting.edit.enable = true Effective on time]

Default value：5

请Based on自己的需求适当调整此值

setting Example

1. SettingsDrag时最多可Show10个Nodes

var setting = {
	edit: {
		enable: true,
		drag: {
			maxShowNodeNum: 10
		}
	}
};
