Function(event, treeId, treeNode)setting.callback.onCollapse

Overview[ Dependency jquery.ztree.core Core js ]

For captureNodes被Collapse的EventsRevert function

IfSettings了 setting.callback.beforeCollapse Methodology，且Back false，将无法Trigger onCollapse EventsRevert function。

Default value：null

Function ParametersAnnotations

eventjs event Object

Standard的 js event Object

treeIdString

Correspond zTree 的 treeId，便于User操控

treeNodeJSON

被Collapse的Nodes JSON DataObject

setting & function Example

1. 每次CollapseNodes后， 弹出该Nodes的 tId、name Information

function zTreeOnCollapse(event, treeId, treeNode) {
    alert(treeNode.tId + ", " + treeNode.name);
};
var setting = {
	callback: {
		onCollapse: zTreeOnCollapse
	}
};
