Function(treeId, treeNodes)setting.callback.beforeDrag

Overview[ Dependency jquery.ztree.exedit Extension js ]

For captureNodes被Drag之前的EventsRevert function，并且Based onBack值Sure是否AllowOpenDragOperation

Default value：null

Function ParametersAnnotations

treeIdString

被Drag的Nodes treeNodes Location zTree 的 treeId，便于User操控

treeNodesArray(JSON)

要被Drag的Nodes JSON DataGather

v3.x AllowMultiple同级Nodes同时被Drag，因此将此ParametersModify为 Array(JSON)

IfDrag时Multiple被Selection的Nodes不是同级关系，则只能DragMouse当前Location位置的Nodes

Back值Boolean

Back值是 true / false

IfBack false，zTree 将终止Drag，也无法Trigger onDrag / beforeDrop / onDrop EventsRevert function

setting & function Example

1. BanAllDragOperation

function zTreeBeforeDrag(treeId, treeNodes) {
    return false;
};
var setting = {
	edit: {
		enable: true
	},
	callback: {
		beforeDrag: zTreeBeforeDrag
	}
