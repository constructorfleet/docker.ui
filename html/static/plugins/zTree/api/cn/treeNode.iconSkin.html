StringtreeNode.iconSkin

Overview[ Dependency jquery.ztree.core Core js ]

Nodes自定义Icon的 className

1、需要Modify css，Increase相应 className 的Settings

2、css ModalitiesSimple、Easy，并且同时Support父NodesExpand、CollapseStatusSwitchPicture

3、css Recommendations采用Picture分割渲染的Modalities以减少反复加载Picture，并且避免Picture闪动

4、zTree v3.x 的 iconSkin 同样Support IE6

5、If想直接Use Picture的UrlPath SettingsNodes的个性化Icon，需要Settings treeNode.icon / treeNode.iconOpen / treeNode.iconClose Properties

Default value：无
