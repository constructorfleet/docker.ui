Booleansetting.edit.drag.isCopy

Overview[ Dependency jquery.ztree.exedit Extension js ]

Drag时, Settings是否AllowCopyNodes。[setting.edit.enable = true Effective on time]

Default value：true

规则Annotations

1、isCopy = true; isMove = true 时，DragNodesPress Ctrl 或 Cmd 键Organisation copy; 否And.. move

2、isCopy = true; isMove = false 时，AllDragOperation都是 copy

3、isCopy = false; isMove = true 时，AllDragOperation都是 move

4、isCopy = false; isMove = false 时，BanDragOperation

setting Example

1. SettingsAllDragOperation都是 copy

var setting = {
	edit: {
		enable: true,
		drag: {
			isCopy: true,
			isMove: false
		}
	}
};
