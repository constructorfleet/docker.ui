Stringsetting.check.radioType

Overview[ Dependency jquery.ztree.excheck Extension js ]

radio 的分组Scope。[setting.check.enable = true 且 setting.check.chkStyle = "radio" Effective on time]

Default value："level"

String FormatAnnotations

radioType = "level" 时，在每一级NodesScope内当做一个分组。
radioType = "all" 时，在整棵树Scope内当做一个分组。

请Attention大小写，Don\'t change

setting Example

1. Settings radio 的判别规And..The whole tree

var setting = {
	check: {
		enable: true,
		chkStyle: "radio",
		radioType: "all"
	}
};
