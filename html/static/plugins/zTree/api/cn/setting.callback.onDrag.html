Function(event, treeId, treeNodes)setting.callback.onDrag

Overview[ Dependency jquery.ztree.exedit Extension js ]

For captureNodes被Drag的EventsRevert function

IfSettings了 setting.callback.beforeDrag Methodology，且Back false，将无法Trigger onDragMove 和 onDrag EventsRevert function。

Default value：null
