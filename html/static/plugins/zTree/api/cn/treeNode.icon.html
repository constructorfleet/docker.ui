StringtreeNode.icon

Overview[ Dependency jquery.ztree.core Core js ]

Nodes自定义Icon的 URL Path。

1、父NodesIf只Settings icon ，会导致Expand、Collapse时都Use同一个Icon

2、父NodesExpand、CollapseUse不同的个性化Icon需要同时Settings treeNode.iconOpen / treeNode.iconClose 两个Properties

3、If想Utilization className Settings个性化Icon，需要Settings treeNode.iconSkin Properties

Default value：无

String Format说明

IconPicture的 url It could be相对Path也It could be绝对Path

Settings相对Path请AttentionPage与Picture之间的关系，确保Picture能够正常加载

treeNode Example

1. SettingsNodes的个性化Icon

var nodes = [
	//父NodesExpand Collapse时Use相同的Icon
	{ name:"父Nodes1", icon:"/img/parent.gif"},

	//父NodesExpand Collapse时分别Use不同的Icon
	{ name:"父Nodes2", iconOpen:"/img/open.gif", iconClose:"/img/close.gif"}, 

	//LeafNodes个性化Icon
	{ name:"LeafNodes", icon:"/img/leaf.gif"}
]
