Booleansetting.edit.enable

Overview[ Dependency jquery.ztree.exedit Extension js ]

Settings zTree 是否处于EditStatus

请在Initialize之前Settings，Initialize后需要改变EditStatus请Use zTreeObj.setEditable() Methodology

Default value: false

Boolean Format说明

true / false Separately Yeah / 不Yeah Edit

EditStatus规则说明

1、点击Nodes时，It won\'t open node.url Assigned URL。
2、全面Support Edit 与 Step Load StatusCoexistence。
3、Yeah对Nodes进行Drag，且SupportBetween trees进行Drag。
4、SupportDrag时 Copy/Move Nodes。（References: setting.edit.drag.isCopy / setting.edit.drag.isMove）
5、YeahPassEditbuttonModify name Properties。
6、YeahPassDeletebuttonDeleteNodes。

请Attention大小写，Don\'t change
