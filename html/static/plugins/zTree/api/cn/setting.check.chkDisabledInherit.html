Booleansetting.check.chkDisabledInherit

Overview[ Dependency jquery.ztree.excheck Extension js ]

当父NodesSettings chkDisabled = true 时，Settings子Nodes是否Auto继承 chkDisabled = true 。[setting.check.enable = true Effective on time]

1、只Use于InitializeNodes时，便于批量Operation。 Yeah已存在的Nodes请Utilization setChkDisabled Methodology单个NodesSettings。

Default value: false

Boolean FormatAnnotations

true Organisation 新Add子Nodes时，Auto继承父Nodes chkDisabled = true 的Properties。

false Organisation 新Add子Nodes时，不继承父Nodes chkDisabled 的Properties。

setting Example

1. 需要子NodesAuto继承 chkDisabled = true

var setting = {
	check: {
		enable: true,
		chkDisabledInherit: true
	}
};
