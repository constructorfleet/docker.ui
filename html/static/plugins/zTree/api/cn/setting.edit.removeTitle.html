String / Function(treeId, treeNode)setting.edit.removeTitle

Overview[ Dependency jquery.ztree.exedit Extension js ]

Deletebutton Title 辅助Information。[setting.edit.enable = true & setting.edit.showRemoveBtn = true Effective on time]

Default value："remove"

String FormatAnnotations

MouseMove到 Deletebutton Up，BrowserAuto弹出的辅助InformationContents，可Based onUser需要自行Modify

Function ParametersAnnotations

treeIdString

Correspond zTree 的 treeId，便于User操控

treeNodeJSON

需要SettingsDeletebutton Title Information的Nodes JSON DataObject

Back值String

Back值同 String Format的Data

setting & function Example

1. SettingsDeletebutton Title 辅助Information为: "DeleteNodes"

var setting = {
	edit: {
		enable: true,
		showRemoveBtn: true,
		removeTitle: "DeleteNodes"
	}
};
......

2. Settings父NodesDeletebutton Title 辅助Information为: "Delete父Nodes"

function setRemoveTitle(treeId, treeNode) {
	return treeNode.isParent ? "Delete父Nodes":"DeleteLeafNodes";
}
var setting = {
	edit: {
		enable: true,
		showRemoveBtn: true,
		removeTitle: setRemoveTitle
	}
};
......
