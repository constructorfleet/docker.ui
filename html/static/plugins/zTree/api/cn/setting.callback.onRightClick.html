Function(event, treeId, treeNode)setting.callback.onRightClick

Overview[ Dependency jquery.ztree.core Core js ]

For capture zTree 上MouseRight点击之后的EventsRevert function

1、IfSettings了 setting.callback.beforeRightClick Methodology，且Back false，将无法Trigger onRightClick EventsRevert function。

2、♪ As long as you will ♪ function 的References赋给 onRightClick Properties，则Right点击 zTree 时，将屏蔽Browser的RightMenu。

Default value：null

Function Parameters说明

eventjs event Object

Standard的 js event Object

treeIdString

Correspond zTree 的 treeId，便于User操控

treeNodeJSON

MouseRight点击时所在Nodes的 JSON DataObject

If不在Nodes上，则Back null

setting & function Example

1. 每次MouseRight点击后， 弹出Mouse所在Nodes的 tId、name Information

function zTreeOnRightClick(event, treeId, treeNode) {
    alert(treeNode ? treeNode.tId + ", " + treeNode.name : "isRoot");
};
var setting = {
	callback: {
		onRightClick: zTreeOnRightClick
	}
};
