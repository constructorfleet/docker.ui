Function(treeNode)zTreeObj.showNode

Overview[ Dependency jquery.ztree.exhide Extension js ]

Show某个被Hide的Nodes。

1、此Functions不Support exedit Extension，因此不要在编辑Status时UseHideNodes的Methodology。

2、Hide/ShowNodes，It affectsNodes的 isFirstNode 和 isLastNode Properties。

3、请Pass zTree Object执行此Methodology。

Function Parameters说明

treeNodeJSON

指定被Show的Nodes JSON Data

请务必Promise此NodesDataObject 是 zTree 内部的DataObject

Back值无

目前无任何Back值

function Example

1. Show某个Hide的Nodes

var treeObj = $.fn.zTree.getZTreeObj("tree");
var node = treeObj.getNodeByParam("isHidden", true);
if (node) {
  treeObj.showNode(node);
}
