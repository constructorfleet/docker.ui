Booleansetting.data.simpleData.enable

Overview[ Dependency jquery.ztree.core Core js ]

Sure zTree Initialize时的NodesData、Step Load时的NodesData、或 addNodes Methodology中输入的 newNodes Data是否采用SimpleData模式 (Array)

No needUser再把Data库中取出的 List Forced conversion to complex JSON 嵌套Format

Default value：false

Boolean Format说明

true / false Separately Use / 不Use SimpleData模式

IfSettings为 true，请务必Settings setting.data.simpleData 内的OtherParameters: idKey / pIdKey / rootPId，并且让Data满足父子关系。

setting Example

1. UseSimple Array Format的Data

var setting = {
	data: {
		simpleData: {
			enable: true,
			idKey: "id",
			pIdKey: "pId",
			rootPId: 0
		}
	}
};
