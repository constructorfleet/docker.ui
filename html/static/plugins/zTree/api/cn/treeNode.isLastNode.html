BooleantreeNode.isLastNode

Overview[ Dependency jquery.ztree.core Core js ]

Records treeNode Nodes是否为同级NodesMedium最后一个Nodes。

Use exhide Extension后，Only forShow的NodesSettings此Properties

InitializeNodesData时，由 zTree Increase此Properties，Don\'t advance the value

Boolean Format说明

true Organisation是同级NodesMedium最后一个Nodes

false Organisation不是同级NodesMedium最后一个Nodes

Nodes被Hide后，isLastNode = false

treeNode Example

1. View当前被选MediumNodes是否是同级NodesMedium最后一个Nodes

var treeObj = $.fn.zTree.getZTreeObj("tree");
var sNodes = treeObj.getSelectedNodes();
if (sNodes.length > 0) {
	var isLastNode = sNodes[0].isLastNode;
}
