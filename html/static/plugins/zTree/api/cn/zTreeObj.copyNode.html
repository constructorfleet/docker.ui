Function(targetNode, treeNode, moveType, isSilent)zTreeObj.copyNode

Overview[ Dependency jquery.ztree.exedit Extension js ]

CopyNodes。

v3.x CopyNodes时Conduct clone Operation。IfYesAccessData在 zTree 内的Object，请Access此Methodology的Back值。

请Pass zTree Object执行此Methodology。

Function ParametersAnnotations

targetNodeJSON

要Copy到的ObjectiveNodes JSON Data

IfCopyTo become根Nodes，请Settings targetNode 为 null Yeah

请务必Promise此NodesDataObject 是 zTree 内部的DataObject

treeNodeJSON

Yes被Copy的NodesData

请务必Promise此NodesDataObject 是 zTree 内部的DataObject

moveTypeString

Copy到ObjectiveNodes的相对位置

"inner"：To become子Nodes，"prev"：To become同级前一个Nodes，"next"：To become同级后一个Nodes

isSilentBoolean

设定CopyNodes后是否AutoExpand父Nodes。

isSilent = true 时，不Expand父Nodes，Other值或缺省Status都AutoExpand。

Back值JSON

Back值是最终Add到 zTree 内的NodesData

Please remember：Back值MediumDataObject 是 treeNode 被 clone After，So it\'s definitely not equal！

function Example

1. 将根Nodes中第二个Nodes CopyTo become 第一个Nodes的子Nodes

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getNodes();
treeObj.copyNode(nodes[0], nodes[1], "inner");

2. 将根Nodes中第二个Nodes CopyTo become 第一个Nodes的前一个Nodes

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getNodes();
treeObj.copyNode(nodes[0], nodes[1], "before");
