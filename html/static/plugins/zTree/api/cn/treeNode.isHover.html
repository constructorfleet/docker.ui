BooleantreeNode.isHover

Overview[ Dependency jquery.ztree.exedit Extension js ]

RecordsNodes 的 hover Status，Mainly for setting.view.addHoverDom / removeHoverDom 。

zTree 内部Use，请勿进行Initialize 或 随意Modify

Default value：false

Boolean Format说明

true OrganisationNodes处于 hover Status

false OrganisationNodes未处于 hover Status
