Function(treeId, treeNodes, targetNode, moveType, isCopy)setting.callback.beforeDrop

Overview[ Dependency jquery.ztree.exedit Extension js ]

For captureNodesDragOperationEnd之前的EventsRevert function，并且Based onBack值Sure是否Allow此DragOperation

Default value：null

如未Drag到有效位置，则Do not trigger此Revert function，直接将NodesRestore原位置

Function ParametersAnnotations

treeIdString

ObjectiveNodes targetNode Location zTree 的 treeId，便于User操控

treeNodesArray(JSON)

被Drag的Nodes JSON DataGather

无论DragOperation为 Copy Still Move，treeNodes 都是当前被DragNodes的DataGather。

targetNodeJSON

treeNodes 被Drag放开的ObjectiveNodes JSON DataObject。

IfDragTo become根Nodes，则 targetNode = null

moveTypeString

指定Move到ObjectiveNodes的相对位置

"inner"：To become子Nodes，"prev"：To become同级前一个Nodes，"next"：To become同级后一个Nodes

isCopyBoolean

DragNodesOperation是 Copy 或 Move

true：Copy；false：Move

Back值Boolean

Back值是 true / false

IfBack false，zTree 将Restore被Drag的Nodes，也无法Trigger onDrop EventsRevert function

setting & function Example

1. Ban将NodesDragTo become根Nodes

function zTreeBeforeDrop(treeId, treeNodes, targetNode, moveType) {
    return !(targetNode == null || (moveType != "inner" && !targetNode.parentTId));
};
var setting = {
	edit: {
		enable: true
	},
	callback: {
		beforeDrop: zTreeBeforeDrop
	}
};
