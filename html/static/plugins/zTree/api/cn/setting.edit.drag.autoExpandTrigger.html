Booleansetting.edit.drag.autoExpandTrigger

Overview[ Dependency jquery.ztree.exedit Extension js ]

Drag时父NodesAutoExpand是否Trigger onExpand EventsRevert function。[setting.edit.enable = true Effective on time]

Default value：false

Boolean Format说明

true / false Separately Trigger / Do not trigger onExpand EventsRevert function。

setting Example

1. SettingsDrag时父NodesAutoExpandTrigger onExpand EventsRevert function

var setting = {
	edit: {
		enable: true,
		drag: {
			autoExpandTrigger: true
		}
	}
};
