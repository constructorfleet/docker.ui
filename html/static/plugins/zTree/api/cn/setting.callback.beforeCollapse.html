Function(treeId, treeNode)setting.callback.beforeCollapse

Overview[ Dependency jquery.ztree.core Core js ]

用于捕获父NodesCollapse之前的EventsRevert function，并且Based onBack值Sure是否AllowCollapseOperation

Default value：null

Function Parameters说明

treeIdString

Correspond zTree 的 treeId，便于User操控

treeNodeJSON

要Collapse的父Nodes JSON DataObject

Back值Boolean

Back值是 true / false

IfBack false，zTree 将不会CollapseNodes，也无法Trigger onCollapse EventsRevert function

setting & function Example

1. 禁止All已Expand的父NodesCollapse

function zTreeBeforeCollapse(treeId, treeNode) {
    return false;
};
var setting = {
	callback: {
		beforeCollapse: zTreeBeforeCollapse
	}
};
