BooleantreeNode.open

Overview[ Dependency jquery.ztree.core Core js ]

Records treeNode Nodes的 Expand / Collapse Status。

1、InitializeNodesData时，If设定 treeNode.open = true，则会直接Expand此Nodes

2、LeafNodes treeNode.open = false

3、为了SolvePart朋友Generate json Data出现的兼容问题, Support "false","true" Character串Format的Data

4、非Step LoadMode，无子Nodes的父NodesSettings open=true 后，可Show为ExpandStatus，但Step LoadMode不会生效。（v3.5.15+）

Default value：false

Boolean Format说明

true OrganisationNodes为 Expand Status

false OrganisationNodes为 Collapse Status

treeNode Example

1. View当前被选MediumNodes的 Expand / Collapse Status

var treeObj = $.fn.zTree.getZTreeObj("tree");
var sNodes = treeObj.getSelectedNodes();
if (sNodes.length > 0) {
	var isOpen = sNodes[0].open;
}
