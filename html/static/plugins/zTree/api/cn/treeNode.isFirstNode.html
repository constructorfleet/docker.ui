BooleantreeNode.isFirstNode

Overview[ Dependency jquery.ztree.core Core js ]

Records treeNode Nodes是否为同级NodesMedium第一个Nodes。

Use exhide Extension后，只针对Show的NodesSettings此Properties

InitializeNodesData时，由 zTree Increase此Properties，Don\'t advance the value

Boolean Format说明

true 表示是同级NodesMedium第一个Nodes

false 表示不是同级NodesMedium第一个Nodes

Nodes被Hide后，isFirstNode = false

treeNode Example

1. View当前被选MediumNodes是否是同级NodesMedium第一个Nodes

var treeObj = $.fn.zTree.getZTreeObj("tree");
var sNodes = treeObj.getSelectedNodes();
if (sNodes.length > 0) {
	var isFirstNode = sNodes[0].isFirstNode;
}
