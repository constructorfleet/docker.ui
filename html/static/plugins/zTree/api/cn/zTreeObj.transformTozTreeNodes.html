Function(simpleNodes)zTreeObj.transformTozTreeNodes

Overview[ Dependency jquery.ztree.core Core js ]

将Simple Array FormatData转换为 zTree Use的Standard JSON 嵌套DataFormat。

Use此Methodology，请务必SettingsNodes唯一标识Properties名称 setting.data.simpleData.idKey 和 父Nodes唯一标识Properties名称 setting.data.simpleData.pIdKey，并且让Data满足父子关系。

请Pass zTree Object执行此Methodology。

Function Parameters说明

simpleNodesArray(JSON) / JSON

To be convertedSimple Array FormatData 或 某个单独的DataObject

Back值Array(JSON)

zTree Use的StandardData，子Nodes都存在于父NodesData的 children Properties中

If simpleNodes It\'s one JSON Object，则被Simple封装成Length为 1 array。

function Example

1. 将Simple Array Format转换为zTreeUse的StandardFormat

var setting = {
	data: {
		simpleData: {
			enable: true,
			idKey: "id",
			pIdKey: "pId",
			rootPId: 0
		}
	}
};
var simpleNodes = [
    {"id":1, "pId":0, "name":"test1"},
    {"id":11, "pId":1, "name":"test11"},
    {"id":12, "pId":1, "name":"test12"},
    {"id":111, "pId":11, "name":"test111"}
];
var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.transformTozTreeNodes(simpleNodes);
