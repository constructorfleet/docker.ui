Boolean / Function(treeId, treeNodes, targetNode)setting.edit.drag.prev

Overview[ Dependency jquery.ztree.exedit Extension js ]

Drag到ObjectiveNodes时，Settings是否AllowMove到ObjectiveNodes前面的Operation。[setting.edit.enable = true Effective on time]

DragObjective是 根 When，Do not trigger prev 和 next，只会Trigger inner

此Functions主要作用是对DragConduct适当限制（辅助Arrow），Need to combine next、inner 一起Use，才能Achieved完整Functions。

Default value：true

Boolean FormatAnnotations

true / false Separately Allow / Not permitted Move到ObjectiveNodes前面

Function ParametersAnnotations

treeIdString

Correspond zTree 的 treeId，便于User操控（多棵树Drag时，是ObjectiveNodesLocation树的 treeId）

treeNodesArray(JSON)

被Drag的Nodes JSON DataGather

targetNodeJSON

Drag时的ObjectiveNodes JSON DataObject

Back值Boolean

Back值同 Boolean Format的Data

setting & function Example

1. BanDrag到Nodes前面的Operation

var setting = {
	edit: {
		enable: true,
		drag: {
			prev: false,
			next: true,
			inner: true
		}
	}
};
......

2. BanDrag到父Nodes前面的Operation

function canPrev(treeId, nodes, targetNode) {
	return !targetNode.isParent;
}
var setting = {
	edit: {
		enable: true,
		drag: {
			prev: canPrev,
			next: true,
			inner: true
		}
	}
};
......
