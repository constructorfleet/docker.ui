Function(treeId, treeNode)setting.callback.beforeMouseDown

Overview[ Dependency jquery.ztree.core Core js ]

For capture zTree 上Mouse按键Press之前的EventsRevert function，并且Based onBack值SureTrigger onMouseDown EventsRevert function

Default value：null

Function ParametersAnnotations

treeIdString

Correspond zTree 的 treeId，便于User操控

treeNodeJSON

Mouse按键Press时LocationNodes的 JSON DataObject

If不在Nodes上，则Back null

Back值Boolean

Back值是 true / false

IfBack false，将Just无法Trigger onMouseDown EventsRevert function，对OtherOperation无任何影响

setting & function Example

1. Ban onMouseDown Events

function zTreeBeforeMouseDown(treeId, treeNode) {
    return false;
};
var setting = {
	callback: {
		beforeMouseDown: zTreeBeforeMouseDown
	}
};
