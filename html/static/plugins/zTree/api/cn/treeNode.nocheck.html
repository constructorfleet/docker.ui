BooleantreeNode.nocheck

Overview[ Dependency jquery.ztree.excheck Extension js ]

1、SettingsNodes是否Hide checkbox / radio [setting.check.enable = true It works]

2、为了SolvePart朋友Generate json Data出现的兼容问题, Support "false","true" Character串Format的Data

Default value：false

Boolean Format说明

true Organisation此NodesDo Not Show checkbox / radio，不影响Check的关联关系，不影响父Nodes的半选Status。

false OrganisationNodes具有正常的CheckFunctions
