Function(parentNode, reloadType, isSilent, callback)zTreeObj.reAsyncChildNodes

Overview[ Dependency jquery.ztree.core Core js ]

强行Step Load父Nodes的子Nodes。[setting.async.enable = true It works]

已经加载过的父Nodes可反复Use此MethodologyReload。

请Pass zTree Object执行此Methodology。

Function Parameters说明

parentNodeJSON

指定需要Step Load的父Nodes JSON Data

1、parentNode = null 时，相当于从根Nodes Root 进行Step Load

2、parentNode.isParent = false 时，不进行Step Load

3、请务必Promise此NodesDataObject 是 zTree 内部的DataObject

reloadTypeString

reloadType = "refresh" 表示Clear后Reload。

reloadType != "refresh" 时，表示Append子Nodes处理。

isSilentBoolean

设定Step Load后是否AutoExpand父Nodes。

isSilent = true 时，不Expand父Nodes，Other值或缺省状态都AutoExpand。

callbackfunction

Refresh完成后的Revert function。

Back值无

目前无任何Back值

function Example

1. 重新Step Load zTree 

var treeObj = $.fn.zTree.getZTreeObj("tree");
treeObj.reAsyncChildNodes(null, "refresh");

2. 重新Step Load当前选Medium第一个Nodes

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getSelectedNodes();
if (nodes.length>0) {
	treeObj.reAsyncChildNodes(nodes[0], "refresh");
}
