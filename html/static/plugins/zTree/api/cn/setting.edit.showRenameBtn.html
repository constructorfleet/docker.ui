Boolean / Function(treeId, treeNode)setting.edit.showRenameBtn

Overview[ Dependency jquery.ztree.exedit Extension js ]

Settings是否ShowEdit Namebutton。[setting.edit.enable = true Effective on time]

当点击某Nodes的Edit Namebutton时：

1、EnterNodesEdit NameStatus。

2、Edit NameOver（Input Lost focus 或 Press Enter 键），会Trigger setting.callback.beforeRename Revert function，User可Based on自己的规则判定是否AllowModify名称。

3、If beforeRename Back false，则继续HoldEdit NameStatus，Until the name fits the rule （Press ESC 键可CancelEdit NameStatus，Restore原名称）。

4、If未Settings beforeRename 或 beforeRename Back true，则EndNodesEdit NameStatus，UpdateNode Name，并Trigger setting.callback.onRename Revert function。

Default value：true

Boolean FormatAnnotations

true / false Separately Show / Hide Edit Namebutton

Function ParametersAnnotations

treeIdString

Correspond zTree 的 treeId，便于User操控

treeNodeJSON

需要Settings是否ShowEdit NamebuttonNodes JSON DataObject

Back值Boolean

Back值同 Boolean Format的Data

setting & function Example

1. Do Not ShowEdit Namebutton

var setting = {
	edit: {
		enable: true,
		showRenameBtn: false
	}
};
......

2. SettingsAll的父NodesDo Not ShowEdit Namebutton

function setRenameBtn(treeId, treeNode) {
	return !treeNode.isParent;
}
var setting = {
	edit: {
		enable: true,
		showRenameBtn: setRenameBtn
	}
};
......
