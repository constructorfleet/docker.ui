BooleantreeNode.check_Focus

Overview[ Dependency jquery.ztree.excheck Extension js ]

用于SettingsNodes的 checkBox / radio 的 focus Status。[setting.check.enable = true It works]

zTree 内部Use，请勿进行Initialize 或 随意Modify

Default value：false

Boolean Format说明

true 表示当前MouseMove到输入框内

false 表示当前MouseMove到输入框外
