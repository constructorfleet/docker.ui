Function(checked)zTreeObj.getCheckedNodes

Overview[ Dependency jquery.ztree.excheck Extension js ]

获取输入框被Check 或 未Check的NodesGather。[setting.check.enable = true It works]

请Pass zTree Object执行此Methodology。

Function Parameters说明

checkedBoolean

checked = true Organisation 被Check 的NodesGather

checked = false Organisation 未Check 的NodesGather

省略此Parameters，Equals true。

Yeah treeNode.nochecked = true 的Nodes不进行获取。

Back值Array(JSON)

Back全部符合要求的NodesGather Array

function Example

1. 获取当前被Check的NodesGather

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getCheckedNodes(true);
