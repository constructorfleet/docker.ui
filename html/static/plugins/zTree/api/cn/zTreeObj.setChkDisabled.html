Function(node, disabled, inheritParent, inheritChildren)zTreeObj.setChkDisabled

Overview[ Dependency jquery.ztree.excheck Extension js ]

Disable 或 Ban 某个Nodes的 checkbox / radio [setting.check.enable = true It works]

1、Nodes的 checkbox / radio 被Disable后，无法Check或CancelCheck，但能够影响父Nodes的半选Status

2、请不要直接Modify已加载Nodes的 treeNode.chkDisabled Properties。

3、请Pass zTree Object执行此Methodology。

Function ParametersAnnotations

treeNodeJSON

YesDisable 或 Ban checkbox / radio 的NodesData

请务必Promise此NodesDataObject 是 zTree 内部的DataObject

disabledBoolean

disabled = true OrganisationDisable checkbox / radio

disabled = false OrganisationBan checkbox / radio

Ignore此Parameters，Equals disabled = false

No impact treeNode.nochecked = true 的Nodes。

inheritParentBoolean

inheritParent = true OrganisationAll父NodesConduct同样的Operation

inheritParent = false OrganisationNo impact父Nodes

Ignore此Parameters，Equals inheritParent = false

inheritChildrenBoolean

inheritChildren = true OrganisationAll子NodesConduct同样的Operation

inheritChildren = false OrganisationNo impact子Nodes

Ignore此Parameters，Equals inheritChildren = false

Back值无

目前无任何Back值

function Example

1. Disable当前选MediumNodes的 checkbox / radio

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getSelectedNodes();
for (var i=0, l=nodes.length; i < l; i++) {
	treeObj.setChkDisabled(nodes[i], true);
}
