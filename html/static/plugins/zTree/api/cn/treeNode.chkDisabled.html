BooleantreeNode.chkDisabled

Overview[ Dependency jquery.ztree.excheck Extension js ]

1、SettingsNodes的 checkbox / radio 是否Disable [setting.check.enable = true It works]

2、为了SolvePart朋友Generate json Data出现的兼容问题, Support "false","true" Character串Format的Data

3、请勿对已加载的NodesModify此Properties，Ban 或 CancelBan 请Use setChkDisabled() Methodology

4、Initialize时，IfYes子孙Nodes继承父Nodes的 chkDisabled Properties，请Settings setting.check.chkDisabledInherit Properties

Default value：false

Boolean FormatAnnotations

true Organisation此Nodes的 checkbox / radio 被Disable。

false Organisation此Nodes的 checkbox / radio YeahUse。

treeNode Example

	{ "id":1, "name":"test1", "checked":true, "chkDisabled":true},
	{ "id":2, "name":"test2", "chkDisabled":true},
