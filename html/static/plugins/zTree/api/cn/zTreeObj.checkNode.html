Function(treeNode, checked, checkTypeFlag, callbackFlag)zTreeObj.checkNode

Overview[ Dependency jquery.ztree.excheck Extension js ]

Check 或 CancelCheck 单个Nodes。[setting.check.enable = true It works]

v3.x 中 checkNode() MethodologyYeahTrigger beforeCheck / onCheck EventsRevert function。便于减少冗余Code

请Pass zTree Object执行此Methodology。

Function ParametersAnnotations

treeNodeJSON

YesCheck 或 CancelCheck 的NodesData

请务必Promise此NodesDataObject 是 zTree 内部的DataObject

checkedBoolean

checked = true OrganisationCheckNodes

checked = false OrganisationNodesCancelCheck

Ignore此Parameters，则Based on对此Nodes的CheckStatusConduct toggle Switch

No impact treeNode.nochecked = true 的Nodes。

checkTypeFlagBoolean

checkTypeFlag = true OrganisationAccording to setting.check.chkboxType PropertiesConduct父子Nodes的Check联动Operation

checkTypeFlag = false Organisation只Modify此NodesCheckStatus，无任何Check联动Operation

checkTypeFlag = false 且 treeNode.checked = checked 时，不会TriggerRevert function，Direct Return

不Influence father and sonNodes中 treeNode.nochecked = true 的Nodes。

callbackFlagBoolean

callbackFlag = true Organisation执行此Methodology时Trigger beforeCheck & onCheck EventsRevert function

callbackFlag = false Organisation执行此Methodology时Do not triggerEventsRevert function

Ignore此Parameters，Equals false

Back值无

目前无任何Back值

function Example

1. Check当前选MediumNodes

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getSelectedNodes();
for (var i=0, l=nodes.length; i < l; i++) {
	treeObj.checkNode(nodes[i], true, true);
}
