Function(treeNode, callbackFlag)zTreeObj.removeNode

Overview[ Dependency jquery.ztree.core Extension js ]

DeleteNodes。

v3.x 中DeleteNodesYeahTrigger beforeRemove / onRemove EventsRevert function。便于减少冗余Code

请Pass zTree Object执行此Methodology。

Function ParametersAnnotations

treeNodeJSON

需要被Delete的NodesData

请务必Promise此NodesDataObject 是 zTree 内部的DataObject

callbackFlagBoolean

callbackFlag = true Organisation执行此Methodology时Trigger beforeRemove & onRemove EventsRevert function

callbackFlag = false Organisation执行此Methodology时Do not triggerEventsRevert function

Ignore此Parameters，Equals false

Back值无

目前无任何Back值

function Example

1. DeleteAll选MediumNodes

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getSelectedNodes();
for (var i=0, l=nodes.length; i < l; i++) {
	treeObj.removeNode(nodes[i]);
}
