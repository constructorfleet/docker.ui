Booleansetting.view.selectedMulti

Overview[ Dependency jquery.ztree.core Core js ]

Settings是否Allow同时选中MultipleNodes。

Default value: true

Boolean Format说明

true / false Separately Support / 不Support 同时选中MultipleNodes

1、Settings为 true时，Press Ctrl 或 Cmd 键可以选中MultipleNodes

2、Settings为 true / false 都不影响Press Ctrl 或 Cmd 键可以让已选MediumNodesUncheck状态（ Uncheck状态可以参考 setting.view.autoCancelSelected ）

setting Example

1. 禁止多点同时选MediumFunctions

var setting = {
	view: {
		selectedMulti: false
	}
};
